# TomeVault Skills Registry

Reusable AI instruction files in `SKILL.md` format, distributed by [TomeVault](https://tomevault.io) and indexed by [Skills.sh](https://skills.sh) and [SkillsMP](https://skillsmp.com).

## Install

Each skill lives in its own subdirectory with a `SKILL.md` and a per-skill README. Copy a skill into your project's `.claude/skills/` (or equivalent) directory.

## Browse

Search, filter, and discover skills across every platform at [tomevault.io](https://tomevault.io).
