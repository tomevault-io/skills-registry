# Research External

Use this skill when local repo context is insufficient and external primary sources are needed.

## Workflow

1. Identify specific unknowns or risky assumptions.
2. Prefer primary sources (official docs/specs).
3. Capture version-aware guidance and constraints.
4. Summarize actionable recommendations with citations.

## Rails Source Priority

For Rails-related questions, prefer in order:

1. Rails Guides / Rails API docs
2. framework-adjacent official docs (Turbo, Stimulus, platform docs)
3. community sources only for supplemental nuance

## Context Discipline

- Research only the unresolved unknowns; avoid broad literature sweeps.
- Stop once recommendations are decision-complete for the active task.
