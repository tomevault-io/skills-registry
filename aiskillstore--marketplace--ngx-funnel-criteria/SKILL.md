# NGX Funnel Criteria Skill

Esta skill proporciona los criterios de evaluación para auditar landing pages, checkouts y flujos de conversión del funnel NGX.

## Cuándo Usar Esta Skill

Activa esta skill cuando necesites:
- Auditar una landing page de NGX
- Evaluar un checkout o flujo de conversión
- Crear o revisar una nueva página del funnel
- Generar reportes de auditoría

## Criterios de Evaluación

### 1. Identidad Visual (20 puntos)

| Elemento | Correcto | Incorrecto | Puntos |
|----------|----------|------------|--------|
| Color primario | #6D00FF | Cyan, colores brillantes | 4 |
| Fondo principal | #0A0A0A | Blanco, grises claros | 4 |
| Cards | Gradiente oscuro + blur | Fondos sólidos | 4 |
| Bordes | rgba(255,255,255,0.05) | Bordes gruesos | 4 |
| Sombras | Múltiples capas + neón | Sombras simples | 4 |

### 2. Claridad de Valor (25 puntos)

| Criterio | Sí | No | Puntos |
|----------|----|----|--------|
| Valor claro en <3 segundos | ✓ | ✗ | 8 |
| Headline confronta problema/deseo | ✓ | ✗ | 6 |
| Subheadline explica solución | ✓ | ✗ | 5 |
| Diferenciador visible | ✓ | ✗ | 6 |

### 3. Copy y Messaging (20 puntos)

| Criterio | Cumple | No cumple | Puntos |
|----------|--------|-----------|--------|
| Usa fórmula CONFRONTA→FUNDAMENTA→RESUELVE | ✓ | ✗ | 5 |
| Tratamiento correcto (tú/usted) | ✓ | ✗ | 4 |
| Menciona agentes correctos | ✓ | ✗ | 3 |
| Sin promesas absolutas | ✓ | ✗ | 4 |
| Sin términos prohibidos | ✓ | ✗ | 4 |

### 4. UX y Usabilidad (20 puntos)

| Criterio | Óptimo | Problema | Puntos |
|----------|--------|----------|--------|
| CTA visible above the fold | ✓ | ✗ | 5 |
| Formulario ≤4 campos | ✓ | ✗ | 4 |
| Mobile responsive | ✓ | ✗ | 5 |
| Touch targets ≥44px | ✓ | ✗ | 3 |
| Navegación clara | ✓ | ✗ | 3 |

### 5. Elementos de Conversión (15 puntos)

| Criterio | Presente | Ausente | Puntos |
|----------|----------|---------|--------|
| CTA claro y específico | ✓ | ✗ | 4 |
| Garantía visible | ✓ | ✗ | 3 |
| Prueba social | ✓ | ✗ | 3 |
| Precio/valor claro | ✓ | ✗ | 3 |
| Eliminación de fricción | ✓ | ✗ | 2 |

## Scoring

- **90-100:** Excelente - Optimizaciones menores opcionales
- **70-89:** Bueno - Algunas mejoras identificadas
- **50-69:** Necesita trabajo - Issues importantes a resolver
- **<50:** Crítico - Problemas bloqueantes que requieren acción inmediata

## Checklist por Tipo de Página

### Landing de Lead Magnet
- [ ] Hook visible inmediatamente
- [ ] Valor del lead magnet claro
- [ ] Formulario simple (email + nombre máximo)
- [ ] Preview del contenido
- [ ] CTA específico ("Descargar gratis", "Acceder ahora")

### Landing de Producto (ASCEND/HYBRID)
- [ ] Propuesta de valor clara
- [ ] Explicación del sistema
- [ ] Diferenciador vs alternativas
- [ ] Pricing transparente
- [ ] Garantía prominente
- [ ] Testimoniales/prueba social
- [ ] FAQ si aplica
- [ ] Múltiples CTAs a lo largo de la página

### Checkout
- [ ] Resumen de compra claro
- [ ] Desglose de precio
- [ ] Métodos de pago visibles
- [ ] Sellos de seguridad
- [ ] Garantía reiterada
- [ ] Soporte disponible
- [ ] Proceso de pasos claro (si multi-step)

## Red Flags

Alertar inmediatamente si:
- 🔴 CTA no visible sin scroll (desktop)
- 🔴 Formulario >5 campos
- 🔴 Tiempo de carga >5 segundos
- 🔴 Links rotos
- 🔴 Errores de consola críticos
- 🔴 Promesas absolutas en copy
- 🔴 Cyan o colores no-NGX prominentes
