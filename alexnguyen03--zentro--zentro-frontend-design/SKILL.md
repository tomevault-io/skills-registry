# Skill 15: Frontend Design

## Purpose

Design practical desktop UI with high clarity and low cognitive load.

## Mandatory Rules

- Prioritize workflow speed over visual novelty.
- Keep hierarchy consistent (typography, spacing, color semantics).
- Ensure keyboard-first productivity for SQL workflows.
- Keep business rules in features/stores, not presentational components.

## Verification

- Manual flow test: connect -> query -> inspect -> export -> history.
- Check desktop resizing behavior and accessibility basics.
