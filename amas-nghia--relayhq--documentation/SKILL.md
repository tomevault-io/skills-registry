# Documentation Skill

## Nguyên tắc

- Viết cho người đọc chưa biết gì về feature này
- Mỗi section trả lời một câu hỏi cụ thể
- Code examples phải chạy được, không phải pseudocode
- Không giải thích WHAT, giải thích WHY và HOW
