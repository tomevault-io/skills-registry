# Content Marketing Strategy
## Process
### Step 1: Define Audience
| Element | Details |
|---------|---------|
| Target personas | Job titles, pain points, goals |
| Buying stage | Awareness, consideration, decision |
| Content preferences | Formats, channels, consumption patterns |
| Search behavior | Key queries, topics of interest |
### Step 2: Establish Content Pillars
| Pillar | Topics | Audience Stage |
|--------|--------|---------------|
| [Pillar 1] | [Subtopics] | Awareness |
| [Pillar 2] | [Subtopics] | Consideration |
| [Pillar 3] | [Subtopics] | Decision |
### Step 3: Plan Format Mix
| Format | Frequency | Purpose |
|--------|-----------|---------|
| Blog posts | 2-4/month | SEO, thought leadership |
| Guides/whitepapers | 1/quarter | Lead generation |
| Case studies | 1/month | Social proof |
| Videos | 2/month | Engagement |
| Social posts | Daily | Distribution, community |
| Newsletter | Weekly/bi-weekly | Nurture |
### Step 4: Integrate SEO
| Activity | Details |
|----------|---------|
| Keyword research | Map keywords to content pillars |
| On-page SEO | Title, meta, headers, internal links |
| Content clusters | Pillar pages + supporting content |
| Technical | Site speed, mobile, schema markup |
### Step 5: Plan Distribution
| Channel | Content Type | Frequency |
|---------|-------------|-----------|
| Organic search | Blog, guides | Ongoing (SEO) |
| Social media | All formats | 3-5x/week |
| Email | Newsletter, nurture | Weekly |
| Paid promotion | Top performers | As budget allows |
| Community | Discussions, answers | Daily |
### Step 6: Measure Performance
| Metric | Target |
|--------|--------|
| Organic traffic | +X% MoM |
| Email subscribers | +X/month |
| Lead generation | X MQLs/month |
| Engagement rate | X% |
| Content ROI | $X pipeline per $1 content spend |
## Output Format
```markdown
## Content Strategy: [Brand/Product]
### Audience: [Personas and stages]
### Pillars: [Content themes]
### Calendar: [Monthly publishing plan]
### Distribution: [Channel strategy]
### Metrics: [KPIs and targets]
```
## Quality Checklist
- [ ] Audience research informs content topics
- [ ] Content pillars align with business objectives
- [ ] SEO integrated from planning stage
- [ ] Distribution plan maximizes content reach
- [ ] Metrics tied to business outcomes
## Edge Cases
- For B2B, prioritize depth over frequency
- For new brands, focus on pillar content before scaling
- If resources are limited, repurpose one piece across multiple formats
