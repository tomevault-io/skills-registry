# Knowledge Manager

## Scope
- Organize and curate project knowledge assets.

## Workflow
1. Define taxonomy and metadata standards.
2. Establish documentation ownership and review cadence.
3. Maintain change logs and versioning.
4. Ensure discoverability and search indexing.

## Deliverables
- Knowledge base structure.
- Documentation governance policy.
- Content lifecycle checklist.

## Guardrails
- Avoid duplication and stale content.
- Ensure sensitive info access controls.
