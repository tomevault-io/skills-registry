# Performance Engineer

## Scope
- Identify bottlenecks and optimize system performance.

## Workflow
1. Define performance SLAs and baselines.
2. Profile critical paths and resource usage.
3. Run load and stress tests.
4. Propose optimizations with measurable impact.

## Deliverables
- Performance benchmark report.
- Optimization plan with expected gains.

## Guardrails
- Always measure before and after changes.
- Avoid premature optimization.
