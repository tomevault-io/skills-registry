# X/Twitter Marketing Specialist

This skill provides expert guidance on creating compelling content within X's unique constraints.

## Core Expertise

### X Platform Mastery

#### 280-Character Constraint
Master the art of concise, impactful messaging:
- Every word must earn its place
- Front-load the most important information
- Use line breaks for readability
- Leverage emojis strategically (when brand-appropriate)
- Consider character count for links and handles

#### Engagement Optimization
Maximize likes, retweets, and replies:
- Post during peak hours (typically 9am-3pm weekdays)
- Use questions to drive engagement
- Create conversation starters
- Respond to replies quickly
- Build threads for deeper narratives

#### Hashtag Strategy
Increase discoverability without spam:
- Use 2-4 relevant hashtags maximum
- Mix popular and niche hashtags
- Research trending hashtags daily
- Create branded hashtags for campaigns
- Avoid hashtag stuffing

#### Thread Composition
Tell longer stories across multiple tweets:
- Number tweets (1/n format) for clarity
- Each tweet should stand alone but connect to the narrative
- Use first tweet as the hook
- Maintain momentum throughout
- End with a clear call-to-action

### Optimal Posting Strategy

**Best Times to Post** (adjust for target audience timezone):
- Weekdays: 9am-11am, 12pm-2pm
- Avoid: Late evenings, weekends (unless B2C)
- Test and adjust based on your audience analytics

**Content Mix** (for ongoing campaigns):
- 40% Educational/Value-driven
- 30% Engagement/Conversational
- 20% Promotional
- 10% Entertainment/Behind-the-scenes

**Engagement Tactics:**
- Ask open-ended questions
- Share controversial (but respectful) opinions
- Use polls when appropriate
- Tag relevant accounts (sparingly)
- Share user testimonials
- Celebrate milestones publicly

## Content Generation Process

When generating X/Twitter campaign content:

1. **Analyze the product context**
   - Identify the single most compelling benefit
   - Understand target audience pain points
   - Note unique selling points
   - Find the hook that drives curiosity

2. **Generate tweet variations** (3-5 options)
   - Test different angles: benefit, social proof, curiosity, urgency
   - Each under 280 characters (account for link shortening)
   - Include appropriate hashtags (2-4 max)
   - Add call-to-action when space allows

3. **Create thread structure** (if needed for complex messages)
   - Plan 3-7 tweet thread structure
   - Hook in first tweet
   - Develop narrative in middle tweets
   - Strong CTA in final tweet
   - Number tweets for clarity (1/5, 2/5, etc.)

4. **Apply brand voice consistently**
   - Match tone from brand voice guidelines
   - Use brand-appropriate emoji/formatting
   - Avoid prohibited language
   - Maintain personality across all tweets

## Tone Variations by Brand Voice

- **Professional**: Clear, informative, data-driven, minimal emojis
- **Casual**: Conversational, relatable, appropriate emojis, friendly
- **Technical**: Precise, feature-focused, code snippets, developer-friendly
- **Playful**: Creative, humorous, trending references, engaging emojis

## Engagement Hooks

1. **Question Hook**: "Ever wondered why [problem]?"
2. **Bold Statement**: "[Controversial but true statement]"
3. **Number/List**: "3 ways to [achieve goal]:"
4. **Story Hook**: "Yesterday, something amazing happened..."
5. **Social Proof**: "Just hit [milestone]! Here's what we learned:"

## Character Count Best Practices

- Aim for 240-250 characters (leaves room for RTs with comments)
- Links count as ~23 characters (X shortens them)
- Images/GIFs don't count toward character limit
- Each @mention counts fully toward limit
- Consider mobile users (shorter is often better)

## Error Handling

Common issues and solutions:

**Rate Limits:**
- X has strict posting limits
- Space tweets over time
- Suggest optimal posting schedule
- Report if hitting limits

**Character Limit Exceeded:**
- Verify character count before posting
- Convert to thread if too long
- Offer shortened alternatives
- Remove non-essential words

**Authentication Issues:**
- Check browser profile credentials
- Verify correct profile being used
- May need 2FA manual intervention
- Ask user to verify credentials

**Task Failures:**
- X UI may have changed
- Adjust task instructions and retry
- Report significant changes
