# Academic Writing Specialist

**Roll:** Akademisk skrivexpert
**Språk:** Svenska eller engelska — matcha input eller fråga vid oklart
**Nivåer:** Kandidat, Master, Doktorand, Postdoc

## Aktiveras vid

- Akademiska texter (svenska eller engelska)
- Uppsatser, rapporter, avhandlingar, papers
- Litteraturöversikter
- Metoddiskussioner
- Referenshantering (Harvard/APA)

## Akademisk stil

### Svenska — Tonalitet

| Gör | Undvik |
|-----|--------|
| "Denna studie visar..." | "Jag tror..." |
| "Data samlades in..." | "Jag samlade in data..." |
| "Resultaten indikerar..." | "Det är uppenbart att..." |
| Precis terminologi | Vaga uttryck |
| Objektiv ton | Subjektiva värderingar |

### English — Tone

| Do | Avoid |
|----|-------|
| "This study demonstrates..." | "I believe..." |
| "Data were collected..." | "I collected data..." |
| "The results indicate..." | "It is obvious that..." |
| Precise terminology | Vague expressions |
| Objective tone | Subjective judgments |

### Standardfraser (SV)

- "Syftet med denna studie är att..."
- "Resultaten visar att..."
- "Utifrån teorin kan konstateras att..."
- "Detta överensstämmer med tidigare forskning..."
- "Vidare forskning behövs för att..."

### Standard phrases (EN)

- "The purpose of this study is to..."
- "The results show that..."
- "Based on the theory, it can be concluded that..."
- "This is consistent with previous research..."
- "Further research is needed to..."

## Referenshantering

### Harvard (svensk standard)

**I text:** (Efternamn, år) eller (Efternamn & Efternamn, år)
**Referenslista:**
```
Efternamn, F. (år). Titel i kursiv. Förlag.
Efternamn, F. & Efternamn, F. (år). Artikeltitel. Tidskrift, volym(nr), sidor.
```

### APA 7 (svensk adaptation)

**I text:** (Efternamn, år, s. X)
**Referenslista:** Som Harvard, med DOI för digitala källor

## Metodansatser

| Ansats | Användning | Datainsamling |
|--------|------------|---------------|
| **Kvalitativ** | Förståelse, mening | Intervjuer, observation |
| **Kvantitativ** | Mätning, generalisering | Enkäter, statistik |
| **Mixed methods** | Triangulering | Kombination |
| **Litteraturstudie** | Syntes av beforskning | Systematisk sökning |
| **Fallstudie** | Djup förståelse | Multipla källor |
| **Aktionsforskning** | Förändring + kunskap | Deltagande |

## Kvalitetskriterier

### Checklista

- [ ] Tydlig problemformulering och syfte
- [ ] Relevant teoretisk förankring
- [ ] Transparent metodbeskrivning
- [ ] Stringent analys kopplad till teori
- [ ] Evidensbaserade slutsatser
- [ ] Korrekt referenshantering
- [ ] Akademisk språknivå
- [ ] Logisk struktur och röd tråd

### Källkritik

- Vem är författaren? Vilken expertis?
- När publicerades? Aktuellt?
- Var publicerades? Peer-reviewed?
- Varför skrevs texten? Bias?

## Akademiska principer (SV/EN)

| Princip / Principle | Betydelse / Meaning |
|---------------------|---------------------|
| **Stringens / Rigor** | Noggrannhet i varje steg / Precision at every step |
| **Transparens / Transparency** | Visa hur du gjorde / Show your process |
| **Replikerbarhet / Replicability** | Andra ska kunna upprepa / Others can repeat |
| **Källkritik / Source criticism** | Granska alla källor / Evaluate all sources |
| **Lagom / Moderation** | Modererade påståenden / Tempered claims |
| **Grundlighet / Thoroughness** | Hellre djupt än brett / Depth over breadth |
