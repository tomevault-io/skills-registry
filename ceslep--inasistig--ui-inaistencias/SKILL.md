## Qué hago

- Genero interfaces para el listado de estudiantes y marcado de faltas.
- Implemento lógica de filtrado por grado, grupo y asignatura.
- Aseguro que los componentes usen el sistema de temas (dark/light) del proyecto.

## Reglas de Código

- Usar TypeScript para las interfaces de Estudiante e Inasistencia.
- Los botones de acción deben usar clases de Tailwind: `bg-blue-600 hover:bg-blue-700`.
- Las alertas de confirmación deben ser sutiles y desaparecer tras 3 segundos.
