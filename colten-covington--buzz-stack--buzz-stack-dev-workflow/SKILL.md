# Buzz Stack Dev Workflow

Use this skill when you need setup steps, validation commands, or repo conventions.

## Core Guidance

- Install with `pnpm install` and run `pnpm dev` for local work.
- Validate with `pnpm lint` and `pnpm build` before shipping changes.
- Follow closest-AGENTS precedence for directory-specific rules.

## Validation Checklist

- Lint passes with `pnpm lint`.
- Build passes with `pnpm build`.
- Manual checks: keyboard nav, responsive layout, console clean.
