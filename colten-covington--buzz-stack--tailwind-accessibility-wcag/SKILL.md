# Tailwind Accessibility and WCAG

Use this skill when building UI with Tailwind and accessibility requirements.

## Core Guidance

- Prefer semantic HTML and add ARIA only when needed.
- Ensure visible focus styles using `focus:ring-2` or similar.
- Avoid `focus:outline-none` without replacement focus styling.
- Use `sr-only` for screen-reader-only labels when needed.
