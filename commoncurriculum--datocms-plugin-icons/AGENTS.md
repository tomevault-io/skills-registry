# Project Notes for Claude

## Package Manager

This project uses **pnpm** as the package manager. Use `pnpm` instead of `npm` for all commands:

- `pnpm install` - Install dependencies
- `pnpm test` - Run tests
- `pnpm build` - Build the project
- `pnpm dev` - Start development server
