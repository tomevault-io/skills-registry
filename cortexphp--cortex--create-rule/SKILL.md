# Create Rule Skill

When the user wants to create a rule, follow these instructions.

## Steps

1. Understand the user's requirements
2. Create a RULE.md file in the appropriate location
3. Add the rule content following the standard format

## Format

Rules should follow this structure:

```markdown
# Rule Name

Description of the rule.

## When to Apply

Conditions for when this rule applies.

## Instructions

Detailed instructions for the rule.
```
