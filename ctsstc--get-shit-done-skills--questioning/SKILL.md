# GSD Questioning Reference

Guidelines for effective user questioning in GSD workflow.

## When to Use

- Clarifying requirements
- Confirming decisions
- Resolving ambiguity

## Question Types

- Clarifying questions
- Confirming questions
- Choice questions
- Elaborating questions

## Best Practices

1. Be specific and focused
2. Provide context
3. Offer options when possible
4. Limit to essential questions
5. Ask one question at a time

## Success Criteria

Questions answered, ambiguity resolved.
