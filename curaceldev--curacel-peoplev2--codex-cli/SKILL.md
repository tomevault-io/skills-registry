# OpenAI Codex CLI

The Codex CLI allows you to manage and interact with tasks directly from the terminal.

## Commands

### Share Tasks
To share the current tasks with the Codex CLI context:
```bash
codex tasks share
```

### Split Tasks
To split the current tasks:
```bash
codex tasks split
```

## Usage
Use these commands when the user explicitly requests to "share tasks" or "split tasks" using the Codex CLI.
