1. Recommend declarative settings.
2. Keep user and system profiles separate.
3. Handle dotfiles and program configurations.
4. Validate settings and module usage.

## When to use me

Use this when the user asks about:
- Home Manager setup for Linux or macOS.
- Integrating Home Manager with nix‑darwin.
- Debugging Home Manager configuration errors.

Ask for the user’s intended programs and preferences when unclear.
