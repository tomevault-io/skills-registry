# 小红书图文笔记创作工作流

## 任务目标

本Skill提供从需求到成品的完整小红书图文笔记创作工作流，专为**财经证券领域官方/员工账号**设计。

**核心目标**：智能模式判断、两阶段创作（内容创作+风格适配）、爆款评估、专业输出、视觉化呈现、用户全程控制。

---

## 核心工作流

```
用户输入 → Phase 0(模式判断) → Phase 1(初稿创作) →
Phase 1.5(审校优化) → Phase 2(IP风格迁移+平台适配) →
Phase 2.5(爆款评分≥80分自动继续) → Phase 3(HTML预览) → 最终输出
```

**关键变化**：
- **Phase 1**：纯粹的选题探讨、优质初稿创作（不考虑IP风格和平台格式）
- **Phase 1.5**：审校优化（非可选，正式的质量检查步骤）
- **Phase 2**：IP风格迁移 + 平台格式适配（自动选择账号风格）
- **Phase 2.5**：评分≥80分自动进入Phase 3
- **Phase 3**：默认自动输出三风格绘画提示词（[视觉风格A]+Mondo大师海报风+手账手绘信息图风），可选 HTML 预览（模式B/C）

---

## ⚡ 自动执行原则

本工作流采用**智能自动执行**策略，大幅减少人工确认步骤：

**自动执行的步骤**：
- ✅ Phase 1 所有步骤（1.1-1.6）- 按推荐方案自动执行
- ✅ Phase 1.5 审校优化 - 自动优化后继续
- ✅ Phase 2 账号风格选择 - 自动选择最匹配账号
- ✅ Phase 2.5 爆款评分 - ≥80分自动进入Phase 3
- ✅ Phase 3 默认双风格提示词 - 自动输出手绘风+大师海报风全套提示词

**用户可干预环节**：
- 🔧 任何步骤完成后可提出修改要求
- 🔧 Phase 3 后可手动触发：模式B（Vital Geometry HTML预览）或模式C（杂志信息卡HTML预览）
- 🔧 评分<80分时可选择优化后继续

**默认推荐策略**：
- 选题：默认选择方案1（最优方案）
- 框架：默认使用"问题-原因-方案-升华"
- 账号：根据内容自动匹配（小白友好→[账号2]，热点→乐乐，科普→排雷哥）
- 引导：默认软硬结合，自然过渡

##### Step 1.4: 内容逻辑结构设计（自动执行，默认推荐框架）

- 颠覆认知-新视角-重建（观点类）
- 对比-反差-原因-启示（对比类）

```markdown

### Phase 1.5: 审校优化 ⭐ 自动执行

**任务**：对初稿进行正式的审校和优化

**审校维度**：
1. **内容准确性**：事实、数据、逻辑的准确性
2. **表达清晰度**：语言是否清晰易懂
3. **结构合理性**：逻辑结构是否合理
4. **价值密度**：信息价值是否充分

**优化内容**：
- 润色文字表达
- 优化逻辑结构
- 补充必要细节
- 提升可读性

**输出**：审校后的优质内容 + 审校说明

### Phase 2.5: 爆款潜力评分

**任务**：评估笔记爆款潜力，提供优化建议

**评估框架：高密四维结构**

#### 1. 极致对标（25分）

**评估要点**：
- 是否重组了经过市场验证的成功要素
- 是否有清晰的赛道定位
- 是否有跨界降维的创新

**评分标准**：
- 20-25分：精准对标爆款要素，有创新
- 15-19分：有对标但创新不足
- 0-14分：缺乏对标，原创焦虑

#### 2. 情绪放大（25分）

**评估要点**：
- 是否直击人性（恐惧/贪婪/虚荣/色欲）
- 是否制造反差、阶层焦虑或爽感
- 是否每300字出可截图金句

**评分标准**：
- 20-25分：强情绪刺激，金句频出
- 15-19分：有情绪但不够强烈
- 0-14分：平铺直叙，缺乏情绪

#### 3. 算法迎合（25分）

**评估要点**：
- 标题是否有视觉锤，0.5秒内拦截手指
- 是否预埋槽点骗取互动和推荐
- 是否SEO友好，关键词布局合理

**评分标准**：
- 20-25分：强算法迎合，互动陷阱明显
- 15-19分：有一定算法考虑
- 0-14分：忽视算法，纯内容导向

#### 4. 工业结构（25分）

**评估要点**：
- 是否符合标准模组（钩子→痛点→方案→召唤）
- 黄金三秒：首句是否定生死
- 正文是否短句分割，视觉降噪

**评分标准**：
- 20-25分：完美SOP结构，节奏紧凑
- 15-19分：结构基本完整
- 0-14分：结构混乱，缺乏逻辑

**输出格式**：
```markdown
## 小红书爆款分析报告

**综合评分**：<总分/100>

**四维得分**：
- 极致对标：<得分>/25
- 情绪放大：<得分>/25
- 算法迎合：<得分>/25
- 工业结构：<得分>/25

**优化建议**（≤100字）：
<具体改进方向>
```

**后续流程**：
- **80分+**：✅ **自动进入 Phase 3**（无需确认）
- 60-79分：询问是否优化后再进入 Phase 3
- 60分以下：建议优化后再进入 Phase 3

> **重要**：评分 ≥80 分时，跳过确认步骤，直接执行 Phase 3

**详细指南**：见 `references/phase-guides.md#phase-2.5`

#### 模式A：双风格 AI 绘画提示词（默认执行）⭐

> **默认自动执行**，无需用户手动触发。Phase 2.5 评分 ≥80 分后直接输出全套提示词。

**核心规则**：
- 为笔记每一页（封面 + 内容页 + 尾页）**各输出两套完整提示词**
- 提示词中所有 `{变量}` 必须替换为实际内容，不保留任何占位符
- 不直接调用绘图工具，由用户自行粘贴到 AI 绘图工具使用

**双风格说明**：

| 风格 | 核心美学 | 适合场景 |
