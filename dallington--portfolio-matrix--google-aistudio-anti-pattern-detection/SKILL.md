# Google AI Studio Anti-Pattern Detection

This skill identifies **low-quality or unsafe patterns** commonly produced by AI-generated frontend code.

## When to use this skill

Use this skill when:
- Code was generated by Google AI Studio
- Components feel bloated or repetitive
- Logic and UI are tightly coupled

## How to use it

1. Scan for AI-generated smells:
   - Huge single files
   - Inline components
   - Logic inside JSX
   - Redundant state
2. Identify unnecessary abstractions
3. Refactor using existing React and Tailwind skills
4. Simplify without changing behavior

## Common Anti-Patterns

- Multiple components in one file
- Overuse of props drilling
- Copy-pasted JSX blocks
- Hardcoded styles and magic values

## Expected Output

- Cleaner, smaller components
- Reduced cognitive load
- Code aligned with senior engineering standards
