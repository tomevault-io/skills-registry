## How to consult Vibe CLI
1. Install Vibe CLI: `curl -LsSf https://mistral.ai/vibe/install.sh | bash`
2. Run the CLI: `vibe` to start interactive mode
3. Query directly: `vibe "your coding task"`
4. Use conversational interface for complex tasks
