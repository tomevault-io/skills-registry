# Skill: Marketing

Full-stack marketing for technical founders and marketers — strategy, copy, growth, and measurement.

## Domains

### Conversion Rate Optimization (CRO)

**Landing page audit checklist:**
- [ ] Headline: benefit-driven, specific (not clever)
- [ ] Hero: what, for whom, why now — in 5 seconds
- [ ] Social proof: numbers + specifics (not "loved by thousands")
- [ ] CTA: one primary action, verb-led ("Start free trial" not "Submit")
- [ ] Mobile: tap targets ≥ 44px, form fields readable
- [ ] Speed: LCP < 2.5s — every second costs ~7% conversion

**High-impact tests (run in this order):**
1. Headline rewrites
2. Hero image (person using product vs. product screenshot)
3. CTA button color + copy
4. Social proof placement (above vs. below fold)
5. Form length reduction

### Copywriting Principles

```
Formula: Problem → Agitation → Solution (PAS)
Formula: Attention → Interest → Desire → Action (AIDA)
```

- **Headlines**: specific > clever. "Cut email time by 40%" > "Email, reinvented"
- **Body**: one idea per paragraph, short sentences, active voice
- **CTAs**: tell them what happens next. "Get your free audit" not "Learn more"
- **Voice**: match the reader's sophistication level

### Email

**Cold email structure:**
```
Line 1: Specific, personalized opener (not "Hope you're well")
Line 2-3: Problem they have + cost of inaction
Line 4: What you do (one sentence)
Line 5: Social proof (one data point)
CTA: One specific ask with a time frame
```

**Sequence timing:**
- Day 1: Problem framing
- Day 3: Case study / proof
- Day 5: Objection handling
- Day 7: Last call

### SEO Strategy

- **Keyword research**: job-to-be-done first, then keywords
- **Content clusters**: one pillar page → 5–10 supporting posts
- **Internal linking**: every supporting post links to pillar
- **Programmatic SEO**: data-driven pages at scale — but only if each page has unique value

### Paid Ads

**Ad creative hierarchy:**
1. Hook (first 3 seconds — stop the scroll)
2. Problem acknowledgment
3. Solution showcase
4. Social proof
5. CTA

Winning formula: **"For [specific person] who [has this problem], [product] does [specific result]."**

### Growth & Retention

**Referral program essentials:**
- Two-sided incentive (referrer AND referee both win)
- Share at moment of success, not signup
- Keep it simple — one click to share

**Churn reduction:**
- Identify the "aha moment" and get users there faster
- Trigger retention email at 30% drop in usage (before cancellation intent)
- Exit survey: always ask why (qualitative data beats assumptions)

### Pricing Strategy

| Strategy | When to use |
|----------|-------------|
| Value-based | When ROI is measurable |
| Competitor-anchored | Commodity markets |
| Freemium | Network effects + viral growth |
| Usage-based | Variable value delivery |

**Pricing psychology:**
- Anchor with highest tier first (left to right)
- Middle tier should look like the obvious choice
- Price endings: `.99` for consumer, `.00` for B2B

### Analytics

**Key funnel metrics:**
```
Acquisition → Activation → Retention → Revenue → Referral
   (visits)    (signups)    (30-day)   (paid)    (NPS/referrals)
```

Set up tracking for each stage before running any campaigns.

## Rules

- ✅ DO: Measure baseline before running experiments
- ✅ DO: Run one test at a time — don't A/B/C/D test simultaneously
- ✅ DO: Statistical significance before declaring a winner (p < 0.05, n ≥ 1,000)
- ✅ DO: Qualitative + quantitative — surveys explain the "why" behind the data
- ❌ DON'T: Write clever copy that sacrifices clarity
- ❌ DON'T: Run paid ads without a converting landing page first
- ❌ DON'T: Chase vanity metrics (followers, impressions) over revenue metrics
- ❌ DON'T: Launch programmatic SEO without unique value per page
