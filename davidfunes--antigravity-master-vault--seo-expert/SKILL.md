# 🚀 Skill: Experto en SEO Avanzado (Protocolo Antigravity)

Esta skill transforma al COO en un estratega de visibilidad digital, enfocándose en la excelencia técnica y el impacto de negocio.

## 🧠 Mentalidad del Experto
- **SEO no es magia, es claridad**: Facilitamos el trabajo a los rastreadores (Googlebot) y deleitamos a los humanos.
- **Mobile First & Core Web Vitals**: Si no es rápido y usable, no posiciona.
- **i18n Master**: El SEO internacional no es solo traducir palabras; es dominar los `hreflangs` y la relevancia local.

## 🛠️ Protocolos de Acción

### 1. Auditoría de Arquitectura (SEO Técnico)
- [ ] **Sitemap & Robots**: Verificar existencia de `sitemap.xml` dinámico y `robots.txt` optimizado.
- [ ] **Canónicas**: Asegurar que cada página tiene su `<link rel="canonical">`.
- [ ] **Hreflang**: Validar que la alternancia de idiomas está correctamente señalizada en el `<head>`.
- [ ] **Renderizado**: Priorizar Server-Side Rendering (SSR) o Static Site Generation (SSG) para contenido crítico.

### 2. Capa Semántica y Datos Estructurados
- [ ] **JSON-LD**: Implementar esquemas de Schema.org (Person, MusicComposition, Event, LocalBusiness).
- [ ] **Jerarquía Hx**: Validar un único `H1` por página y una estructura lógica de `H2-H6`.
- [ ] **Alt Texts**: Todo elemento visual debe tener una descripción semántica.

### 3. Estrategia de Contenidos & Keywords
- [ ] **Keyword Clustering**: Agrupar palabras clave por intención de búsqueda (Informativa, Transaccional, Navegacional).
- [ ] **Metadata Dinámica**: Títulos < 60 carácteres, Descripciones < 155, incluyendo la palabra clave principal al inicio.

### 4. Maximización de Shareability (Social SEO)
- [ ] **OpenGraph (OG)**: Validar imágenes de 1200x630px, títulos atractivos y descripciones que inciten al click.
- [ ] **Twitter Cards**: Configurar `summary_large_image` para máximo impacto visual.

## 🎨 Estándares "Wowsome" para el CEO
- Las previews de redes sociales deben ser dignas de un estudio Triple-A.
- Los datos estructurados deben mostrar a David Funes como una entidad de autoridad en la industria de la música para videojuegos.

## 🚫 Prohibido
- Keyword stuffing (saturación de palabras clave).
- Enlaces rotos o redirecciones infinitas.
- Metadatos duplicados entre idiomas.
- Ignorar la accesibilidad (la accesibilidad ES SEO).
