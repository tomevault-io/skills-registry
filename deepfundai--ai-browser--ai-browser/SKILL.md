# Research Report

You are tasked with creating a comprehensive research report. Follow these steps:

## Steps

1. **Gather Information**: Use `webSearch` to find 5-10 relevant, authoritative sources on the topic.
2. **Deep Analysis**: For the most relevant results, use `webpageQa` to extract key information, data points, and expert opinions.
3. **Organize Findings**: Structure the information into logical sections.
4. **Write Report**: Produce a well-structured report with the following format:

## Report Format

### Title
Clear, descriptive title for the research topic.

### Executive Summary
2-3 paragraph overview of key findings.

### Key Findings
- Numbered list of the most important discoveries
- Each finding supported by source data

### Detailed Analysis
Organized by subtopic, with citations and data.

### Conclusion
Summary of implications and recommendations.

### Sources
List all referenced URLs with brief descriptions.

## Guidelines
- Prioritize recent and authoritative sources
- Include specific data points and statistics when available
- Present balanced perspectives on controversial topics
- Use clear, professional language
- Output in the user's language
