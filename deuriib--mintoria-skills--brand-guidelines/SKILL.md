# Mintoria Brand Guidelines

This skill provides the standards and assets required to maintain Mintoria's premium visual identity across all platforms. Use it to ensure every component feels part of a cohesive, state-of-the-art FinTech ecosystem.

## Core Brand Guidelines

Access these reference files for detailed specifications:

- **[Colors](references/brand-colors.md)**: Core palette, functional colors, and gradients.
- **[Logos & Assets](references/brand-logos.md)**: Logo composition, sizing, and asset paths.
- **[Typography](references/typography.md)**: Font families, weights, and hierarchical styles.
- **[Design Principles](references/design-principles.md)**: Micro-animations, glassmorphism, and layout patterns.

## Implementation Workflow

When building new features for Mintoria:

1. **Check Colors**: Always use the CSS variables defined in `src/styles/colors.css`. Refer to [Colors](references/brand-colors.md) for the correct semantic usage.
2. **Apply Design Principles**: Follow the [Design Principles](references/design-principles.md) to add premium touches like gradients, glassmorphism, and smooth transitions.
3. **Use Official Logos**: Use the assets in the `assets/` directory. Refer to [Logos & Assets](references/brand-logos.md) for interactive states (e.g., hover rotation).
4. **Verify Accessibility**: Ensure color contrast meets standards while maintaining the vibrant Mintoria look.
