

# Application Overview

This is a 3D web-based game built with React, TypeScript, and Three.js. The game is called "Mars Terraforming". The objective of the game is for the player to terraform Mars by building structures and gathering resources.

## Technical Stack

The stack includes:

- React for building the user interface
- TypeScript for adding static typing and improving code quality
- Three.js for rendering 3D graphics
- Node.js tsoa as backend in the future

## Coding Guidelines

Here are some guidelines for coding in this project:

- Use the latest version of TypeScript and follow the TypeScript language guidelines.
- Use React hooks and services to manage state and side effects.
- Follow the principles of clean code and clean architecture.

## Project Structure

The project is structured using the principles of clean architecture. The main directories include:

- `src`: contains the source code for the application
- `public/models`: contains 3d models
- `public/textures`: contains textures
- `src/ui`: contains React components for the user interface

App modules have self named directories with their own components, hooks, and services for example `src/app/start`
