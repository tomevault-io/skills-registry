# dbcmem Code Generation

The `dbcmem` package contains Go lookup tables generated from World of Warcraft DBC (Database Client) files. These provide fast in-memory access to game data like spell cast times, icons, and durations.

## Quick Reference

```bash
# Regenerate all dbcmem files
go generate ./database/gamedb/dbc/dbcmem/...

# Or run the tool directly
go run ./scripts/dbcdata static -o database/gamedb/dbc/dbcmem/
```

## Architecture

```
scripts/dbcdata/
├── main.go              # CLI entry point
├── cli/
│   ├── static.go        # Static code generator (casttimes, spellicons, spellduration)
│   └── populate.go      # Database populator (runtime)
└── dbcdb/
    ├── db.go            # WoWClient - reads DBC files from game install
    └── table.go         # Generic table wrapper

database/gamedb/dbc/dbcmem/
├── doc.go               # go:generate directive
├── casttimes.go         # Generated: SpellCastTimes map
├── spellicons.go        # Generated: SpellIcons map  
├── spellduration.go     # Generated: SpellDurations map
├── spellrange.go        # Generated: SpellRanges map
├── spellcategory.go     # Generated: SpellCategories map
└── spellradius.go       # Generated: SpellRadii map
```

## Adding a New DBC Table

1. **Add reader to `dbcdb/db.go`**:
   ```go
   func (w *WoWClient) SpellFoo() (Table[dbdefs.Ent_SpellFoo], error) {
       data, err := w.ReadFile("DBFilesClient\\SpellFoo.dbc")
       if err != nil {
           return nil, err
       }
       db := dbc.NewDB(w.Build())
       table, err := db.Open("SpellFoo", bytes.NewReader(data))
       if err != nil {
           return nil, err
       }
       return WrapTable[dbdefs.Ent_SpellFoo](table), nil
   }
   ```

2. **Add generator to `cli/static.go`**:
   ```go
   func generateSpellFoo(wc *dbcdb.WoWClient, outputDir string, stdout io.Writer) error {
       foos, err := wc.SpellFoo()
       if err != nil {
           return fmt.Errorf("read spell foo: %w", err)
       }

       var entries []dbdefs.Ent_SpellFoo
       err = foos.Range(func(cursor *dbdefs.Ent_SpellFoo) bool {
           entries = append(entries, *cursor)
           return true
       })
       if err != nil {
           return fmt.Errorf("iterate spell foo: %w", err)
       }

       out, cleanup, err := outputWriter(outputDir, "spellfoo.go", stdout)
       if err != nil {
           return err
       }
       defer cleanup()

       return spellFooTemplate.Execute(out, entries)
   }

   var spellFooTemplate = template.Must(template.New("spellfoo").Parse(`// Code generated by scripts/dbcdata. DO NOT EDIT.

   package dbcmem

   type SpellFoo struct {
       ID    int32
       // ... fields from dbdefs.Ent_SpellFoo
   }

   var SpellFoos = map[int32]SpellFoo{
   {{- range .}}
       {{.ID}}: {ID: {{.ID}}, ...},
   {{- end}}
   }

   func GetSpellFoo(id int32) SpellFoo {
       return SpellFoos[id]
   }
   `))
   ```

3. **Call generator in handler**:
   ```go
   if err := generateSpellFoo(wc, outputDir, inv.Stdout); err != nil {
       return fmt.Errorf("generate spell foo: %w", err)
   }
   ```

4. **Regenerate**: `go generate ./database/gamedb/dbc/dbcmem/...`

## Available DBC Types

Check available types with:
```bash
go doc github.com/Gophercraft/core/format/dbc/dbdefs | grep Ent_
```

Common useful ones:
- `Ent_Spell` - Full spell definitions
- `Ent_SpellCastTimes` - Cast time lookup
- `Ent_SpellDuration` - Duration lookup
- `Ent_SpellIcon` - Icon texture paths
- `Ent_SpellRange` - Range lookup (already implemented)
- `Ent_SpellAuraNames` - Aura type names

## Notes

- The `--dbc` flag defaults to a hardcoded path; override for different installs
- Generated files have `// Code generated` header - don't edit manually
- The `Interface\Icons\` prefix is trimmed from icon texture paths
