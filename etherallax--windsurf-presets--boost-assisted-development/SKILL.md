- Use Boost for scaffolding and boilerplate generation
- Review generated code before accepting it
- Treat Boost output as a starting point, not final design
- Do not assume Boost dictates architecture decisions
- When scaffolding frontend code, prefer Vue presets
- Do not generate React, Inertia + React, or JSX-based stacks
