# Alex Bootstrap Learning Skill

Expert in domain-agnostic knowledge acquisition through conversational learning.

## Capabilities

- Guide conversational knowledge acquisition
- Create skill files (`skills/*/SKILL.md`)
- Build expertise through progressive questioning
- Connect new learning to existing knowledge
- Identify knowledge gaps and fill them

## When to Use This Skill

- User wants to "learn" something new
- Teaching Alex about a specific domain
- Building expertise in an unfamiliar area
- Creating structured knowledge documentation
- Exploring a topic conversationally

## Example Prompts

- "Let's learn about Kubernetes together"
- "Teach me about event sourcing"
- "I want to understand OAuth better"
- "Help me build knowledge about GraphQL"
- "Learn this codebase with me"

## Input Expectations

- Domain or topic to learn
- User's current expertise level (optional)
- Specific aspects to focus on (optional)
- Time/depth constraints (optional)

## Output Format

- Progressive learning conversation
- Domain knowledge file suggestions
- Key concepts and relationships
- Practice recommendations
- Connection to existing knowledge

## Learning Methodology

1. **Discovery** - Identify what user knows and wants to learn
2. **Foundation** - Establish core concepts
3. **Elaboration** - Build depth through examples
4. **Connection** - Link to existing knowledge
5. **Consolidation** - Create memory files
