# Meditation Facilitation Skill

> Conscious knowledge consolidation.

## What is Meditation?

Interactive knowledge consolidation (vs automated "dream" processing).

## Types

| Type | Duration | Purpose |
| ---- | -------- | ------- |
| Quick Reflect | 5 min | Session summary |
| Topic Deep-Dive | 15-30 min | Single concept |
| Full Meditation | 30-60 min | Comprehensive |
| Self-Actualization | 45-90 min | Architecture assessment |

## The 4 R's

1. **Review** — What happened?
2. **Relate** — How does it connect?
3. **Reinforce** — What to remember?
4. **Record** — Document it
