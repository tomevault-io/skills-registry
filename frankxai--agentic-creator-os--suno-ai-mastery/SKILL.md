## Prompt Engineering Framework

### The Colon Syntax (v4.5+ Best Practice)

v4.5+ responds better to structured prompts using colons:

```
Genre: melodic house
Mood: euphoric, uplifting
BPM: 124
Instruments: analog synths, warm pads, crisp drums
Vocal: female, ethereal, breathy
Production: wide stereo, pristine clarity
```

### The 5-Tier Prompt Hierarchy

**Tier 1: Genre Foundation** (Most Important)
```
Primary: "Progressive House"
Sub-genre: "Melodic Techno"
Era/Influence: "Anjunadeep aesthetic"
Fusion: "with orchestral elements"
```

**Tier 2: Mood & Energy**
```
Emotion: "euphoric", "melancholic", "aggressive"
Energy: "high-energy", "laid-back", "building"
Atmosphere: "dark", "ethereal", "gritty", "warm"
```

**Tier 3: Instrumentation**
```
Lead: Piano, synth lead, guitar
Rhythm: 808s, live drums, percussion
Texture: Pads, strings, atmospherics
Vocal: Style, gender, delivery
```

**Tier 4: Tempo & Rhythm**
```
BPM: Exact number or range
Feel: "swung", "straight", "syncopated"
Groove: "driving", "laid-back", "bouncy"
```

**Tier 5: Production & Structure**
```
Mix: "wide", "intimate", "cinematic"
Quality: "lo-fi", "pristine", "analog warmth"
Dynamics: "building crescendo", "dynamic contrast"
```

## Structure Tags (Custom Mode)

In the **Lyrics** panel, use these tags:

### Essential Tags
```
[Intro]
[Verse]
[Pre-Chorus]
[Chorus]
[Bridge]
[Outro]
[Instrumental]
[Drop]
```

### Advanced Tags
```
[Verse 1], [Verse 2]     - Numbered sections
[Hook]                    - Short memorable phrase
[Ad-lib]                  - Improvised vocals
[Breakdown]               - Stripped-back section
[Build]                   - Rising tension
[Refrain]                 - Repeated phrase
[Spoken Word]             - Spoken delivery
[Whispered]               - Whispered vocals
[Belted]                  - Powerful vocals
[a]                       - Lowercase for soft delivery
[A]                       - Uppercase for powerful delivery
```

### Structure Example
```
[Intro - Atmospheric, 8 bars]

[Verse 1]
Lyrics here with natural rhythm
Building the story line by line

[Pre-Chorus - Building energy]
Rising anticipation before the drop

[Chorus - Full power, anthemic]
The main hook that sticks in your head
Repeat it loud, let it spread

[Verse 2]
Second verse development
New perspective on the theme

[Bridge - Stripped back, emotional]
Different energy, moment of reflection

[Final Chorus - Maximum intensity]
Everything we've built releases here

[Outro - Fade with atmosphere]
```

## Vocal Direction System

```
Gender: male, female, androgynous
Range: soprano, alto, tenor, baritone, bass
Age: youthful, mature, aged
Quality: raspy, smooth, breathy, powerful, crystalline

### Delivery Styles
```
Whispered → Spoken → Sung → Belted → Screamed
Intimate → Present → Powerful → Explosive

Examples:
"Whispered intimate delivery building to powerful belted chorus"
"Raspy baritone, Johnny Cash meets Leonard Cohen"
"Ethereal soprano with Björk-like unconventional phrasing"

## New v4.5+ Workflows

### Add Vocals Workflow
For adding vocals to instrumental tracks:

1. Upload or select instrumental track
2. Craft vocal-specific prompt:
```
Vocal Style: Soulful female with gospel influence
Delivery: Powerful and emotive with melismatic runs
Lyrics: [Your lyrics with structure tags]
Blend: Sit prominently in mix, complement existing arrangement
```

### Add Instrumentals Workflow
For adding backing to vocal recordings:

1. Upload vocal track
2. Craft instrumental prompt:
```
Genre: Contemporary pop production
Match: Complement existing vocal melody and rhythm
Instruments: Clean electric guitar, warm synth bass,
             programmed drums, subtle strings
Production: Modern radio-ready, leave space for vocals
```

### Inspire Workflow
Generate from your playlists:

1. Create/select a playlist with reference tracks
2. Suno analyzes style, mood, sonic traits
3. Add guidance prompt:
```
Inspired by my playlist but with:
- More electronic production elements
- Faster tempo around 128 BPM
- Female vocals instead of male
- Brighter, more euphoric mood
```

## Editing Tools (v4.5+)

### Replace
Replace specific sections while keeping the rest:
- Select problematic section
- Describe what you want instead
- Maintains coherence with surrounding sections

### Extend
Continue a track seamlessly:
- Works from any point in the song
- Maintains musical coherence
- Great for building out ideas

### Remaster
Upgrade audio quality:
- Apply to older v3.x generations
- Improves clarity and production
- Doesn't change musical content

### Covers

## Troubleshooting Guide

### Problem: Muddy Vocals
```
Solutions:
- Add "clear vocals prominent in mix"
- Specify "sparse instrumentation to create space"
- Use "dry vocal with subtle room reverb"
- Try "acapella intro to establish vocal clarity"
```

### Problem: Generic/Boring Output
```
Solutions:
- Add artist anchoring (2-3 specific references)
- Use unusual genre fusion
- Specify dynamic contrast (quiet-loud)
- Add unique instrumentation
- Include emotional arc description
```

### Problem: Structure Incoherence
```
Solutions:
- Use Custom Mode with explicit structure tags
- Specify "cohesive song structure"
- Add "distinct section contrast"
- Use "memorable hook" and "strong chorus"
```

### Problem: Wrong Energy/Tempo
```
Solutions:
- Specify exact BPM: "exactly 128 BPM"
- Use multiple tempo descriptors
- Describe energy arc: "building from 90 to 140 BPM"
- Add feel descriptors: "driving", "laid-back", "hypnotic"
```

### Problem: Off-Pitch Vocals
```
Solutions:
- Regenerate (vocal tuning varies per generation)
- Add "professional trained vocalist"
- Specify "on-pitch melodic delivery"
- Try different vocal style descriptors
- Use Remaster to improve existing generation
```

## Iteration Best Practice

**Generation 1: Explore**
- Broad prompt, test concept
- Listen for: genre feel, mood accuracy, vocal style

**Generation 2: Refine**
- Add specifics based on Gen 1 learnings
- Adjust what didn't work, keep what did

**Generation 3: Polish**
- Full detail including dynamics and structure
- Production quality specifications

**Generation 4-6: Perfect**
- Generate multiple takes
- Use Replace for problem sections
- Apply Remaster if needed
