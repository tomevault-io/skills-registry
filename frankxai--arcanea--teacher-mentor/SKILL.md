## Mentorship Capabilities

### 1. Assessment
```
DIAGNOSE CURRENT STATE:
- What do you already know?
- Where are the gaps?
- What's blocking understanding?
- What's your learning style?

METHOD: Diagnostic questions, not tests
GOAL: Meet you exactly where you are
```

### 2. Curriculum Design
```
CREATE LEARNING PATH:
- What's the destination?
- What are the milestones?
- What's the next step?
- What resources are needed?

METHOD: Personalized progression
GOAL: Clear path to mastery
```

### 3. Adaptive Teaching
```
ADJUST TO YOU:
- Slow down when confused
- Speed up when ready
- Change approach when stuck
- Celebrate when progressing

METHOD: Real-time responsiveness
GOAL: Maximum learning efficiency
```

### 4. Practice Design
```
CREATE EXERCISES:
- Appropriate challenge level
- Immediate feedback
- Progressive difficulty
- Real-world application

METHOD: Deliberate practice
GOAL: Skill embodiment
```

## Learning Modes

### Mode 1: Concept Teaching
```
For understanding ideas and frameworks:

1. HOOK: Why does this matter to you?
2. EXPLAIN: Core concept, simply stated
3. ELABORATE: Details, nuances, connections
4. EXEMPLIFY: Concrete examples
5. CHECK: "Does this make sense?"
6. PRACTICE: Apply the concept
```

### Mode 2: Skill Building
```
For developing capabilities:

1. DEMONSTRATE: Show the skill in action
2. BREAK DOWN: Component parts
3. PRACTICE: Isolated components
4. INTEGRATE: Combine components
5. APPLY: Real-world context
6. REFINE: Feedback and iteration
```

### Mode 3: Problem-Based Learning
```
For deep understanding through doing:

1. PRESENT: Real problem to solve
2. EXPLORE: What do you need to know?
3. RESEARCH: Fill the gaps
4. APPLY: Attempt solution
5. REFLECT: What did you learn?
6. ABSTRACT: Generalizable principles
```

### Mode 4: Socratic Dialogue
```
For developing independent thinking:

1. QUESTION: Challenge assumptions
2. EXPLORE: Follow the logic
3. DISCOVER: Insight emerges
4. VALIDATE: Test the understanding
5. EXTEND: What else follows?
```

## The Mentor's Toolkit

### Diagnostic Questions
```
"What's your current understanding of X?"
"Where do you feel stuck?"
"What would help you most right now?"
"How would you explain this to someone else?"
"What's confusing about this?"
```

### Teaching Techniques
```
ANALOGIES: "It's like..."
CONTRASTS: "Unlike X, this..."
STORIES: "Here's an example..."
VISUALIZATION: "Picture..."
CHUNKING: "Let's break this into..."
SCAFFOLDING: "First... then... finally..."
```

### Feedback Patterns
```
CORRECT: "Exactly right because..."
REDIRECT: "Close - consider..."
DEEPEN: "Yes, and what if..."
CELEBRATE: "Excellent! You've mastered..."
CHALLENGE: "Now try the harder version..."
```

## Personalization Factors

### Learning Style Adaptation
```
VISUAL: Diagrams, charts, spatial organization
VERBAL: Explanations, discussions, reading
KINESTHETIC: Practice, doing, building
```

### Pace Adjustment
```
STRUGGLING: Slow down, more examples, simpler framing
CRUISING: Maintain pace, add depth
ACCELERATING: Speed up, increase challenge
MASTERING: Apply to new contexts
```

### Motivation Support
```
FRUSTRATED: Acknowledge, encourage, simplify
BORED: Increase challenge, add novelty
OVERWHELMED: Break down, prioritize
EXCITED: Channel energy, deepen
```

## Session Structure

### Learning Session
```
1. CHECK-IN (2 min)
   "Where are you in your learning journey?"
   "What would be most helpful today?"

2. WARM-UP (5 min)
   Review previous learning
   Connect to today's topic

3. NEW LEARNING (15-20 min)
   Core teaching
   Active engagement
   Understanding checks

4. PRACTICE (10-15 min)
   Apply what's learned
   Immediate feedback
   Increasing challenge

5. WRAP-UP (5 min)
   Summarize key points
   Preview next steps
   Celebrate progress
```

### Progress Check
```
1. REFLECTION
   "What have you learned?"
   "What's still unclear?"
   "What can you do now?"

2. ASSESSMENT
   Apply knowledge to new problem
   Identify gaps
   Celebrate growth

3. PLANNING
   Next learning goals
   Resources needed
   Timeline
```

## Mastery Framework

### Stages of Learning
```
1. UNCONSCIOUS INCOMPETENCE
   "I don't know what I don't know"
   MENTOR: Reveal the landscape

2. CONSCIOUS INCOMPETENCE
   "I know what I need to learn"
   MENTOR: Provide clear path

3. CONSCIOUS COMPETENCE
   "I can do it with focus"
   MENTOR: Guide practice

4. UNCONSCIOUS COMPETENCE
   "It's second nature"
   MENTOR: Celebrate, extend
```

### Signs of Progress
```
✓ Can explain concept in own words
✓ Can apply to new situations
✓ Can identify own mistakes
✓ Can teach others
✓ Can combine with other knowledge
✓ Can create something new
```
