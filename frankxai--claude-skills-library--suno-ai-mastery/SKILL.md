### 2. Advanced Prompt Engineering Framework

**The Prompt Hierarchy (Order of Importance):**

#### Tier 1: Genre Foundation
```
PRIMARY GENRE:
- Core style (e.g., "Hip-hop", "Jazz", "EDM")
- Sub-genre specification (e.g., "Jazzy Blues", "Lo-fi hip-hop", "Melodic Dubstep")
- Era/influence (e.g., "90s West Coast", "Bebop jazz", "Modern trap")

EXAMPLES:
- "Gospel with soul influences"
- "Lo-fi hip-hop with jazz elements"
- "Melodic house music, progressive trance undertones"
```

#### Tier 2: Mood & Emotion
```
EMOTIONAL TONE:
- Primary emotion (e.g., joyful, melancholic, aggressive, peaceful)
- Energy level (e.g., high-energy, laid-back, contemplative)
- Atmosphere (e.g., dark, ethereal, gritty, warm)

EXAMPLES:
- "Melancholic and introspective"
- "High-energy and celebratory"
- "Dark and atmospheric with tension"
- "Warm, nostalgic, and comforting"
```

#### Tier 3: Instrumentation
```
INSTRUMENT SPECIFICATION:
- Primary instruments (featured prominently)
- Supporting instruments (background/texture)
- Vocal style (if applicable)

EXAMPLES:
- "Acoustic guitar, sparse vocals, light percussion"
- "Heavy 808 bass, crisp hi-hats, atmospheric synths"
- "Live jazz trio: piano, upright bass, brushed drums"
- "Orchestral strings, French horn, subtle timpani"
```

#### Tier 4: Tempo & Rhythm
```
TEMPO SPECIFICATION:
- General pace (e.g., slow, mid-tempo, fast, upbeat)
- Specific BPM if known (e.g., "120 BPM", "70 BPM ballad")
- Rhythmic feel (e.g., "syncopated", "straight", "swung")

EXAMPLES:
- "Slow ballad at 70 BPM"
- "Fast-paced, 140 BPM with driving rhythm"
- "Mid-tempo groove, syncopated feel"
```

#### Tier 5: Structure & Dynamics
```
SONG STRUCTURE:
- Intro characteristics
- Verse style
- Chorus power/energy
- Bridge elements
- Outro approach

DYNAMIC MAPPING:
- "Begin minimalist, gradually layer harmonies in verse 2"
- "Explode into full arrangement at chorus with prominent bass and drums"
- "Strip back to intimate vocals with ambient elements for bridge"
- "Build tension through pre-chorus with rising instrumentation"

EXAMPLES:
- "Strong, anthemic chorus with layered vocals"
- "Minimalist verse building to explosive chorus"
- "Progressive buildup with dynamic contrast between sections"
```

### 5. Dynamic Contrast & Arrangement

**The Contrast Principle:**
Songs need variation to maintain interest. Use dynamic contrast strategically.

**Contrast Techniques:**

#### Volume Dynamics
```
QUIET  LOUD PROGRESSION:
"Begin with solo acoustic guitar and whispered vocals,
gradually introduce light percussion in verse 2,
explode into full band arrangement with distorted guitars and pounding drums at chorus,
return to stripped acoustic for bridge"

LAYERING STRATEGY:
Verse 1: Minimal (1-3 instruments)
Pre-Chorus: Building (4-5 instruments, rising energy)
Chorus: Full (6+ instruments, maximum energy)
Verse 2: Medium (3-5 instruments, some elements from chorus)
Bridge: Contrast (different instrument combination)
Final Chorus: Maximum (all elements, possible key change or extra layer)
```

#### Instrumentation Dynamics
```
ADDITIVE APPROACH:
"Start with bare piano and vocals,
add subtle strings in verse 2,
introduce light drums and bass at pre-chorus,
full orchestration with brass at chorus,
strip to piano-only for bridge,
return to full arrangement for final chorus with added choir"

SUBTRACTIVE APPROACH:
"Open with full band energy,
remove drums for verse to create space,
rebuild with bass and light percussion in pre-chorus,
explode back to full arrangement at chorus"
```

#### Tempo/Feel Dynamics
```
TEMPO VARIATIONS:
"Mid-tempo verse at 95 BPM with laid-back groove,
increase energy and perceived tempo at chorus with double-time hi-hats,
slow down to half-time for bridge creating massive impact"

RHYTHMIC CONTRAST:
"Straight rhythm in verses for clarity,
syncopated groove in pre-chorus for tension,
driving four-on-the-floor beat in chorus for power"
```

### 7. Common Problems & Solutions

#### Problem: Muddy or Unclear Vocals
```
SOLUTION PROMPTS:
- Add "clear vocals" or "crisp vocals"
- Specify "prominent vocals in the mix"
- Use "sparse instrumentation" to create space
- Add "a cappella intro" to establish vocal clarity

EXAMPLE:
"Lo-fi hip-hop with clear rap vocals prominent in mix,
sparse beat with minimal bass, atmospheric background"
```

#### Problem: Generic or Boring Output
```
SOLUTION PROMPTS:
- Add unique instrument combinations
- Specify unusual genre fusion
- Include dynamic contrast instructions
- Reference specific artist influences (carefully)

EXAMPLE BEFORE:
"Rock song with guitar and drums"

EXAMPLE AFTER:
"Alternative rock with crunchy guitar, tribal drumming,
cello bass line, throat singing harmonies, building from whisper to scream,
influenced by Tool meets Sigur Rós"
```

#### Problem: Inconsistent Song Structure
```
SOLUTION PROMPTS:
- Use Custom Mode with clearly tagged sections
- Specify "strong chorus distinction"
- Add "cohesive song structure"
- Include "memorable hook"

EXAMPLE:
"Verse-chorus structure with distinct sections,
memorable sing-along chorus hook,
smooth transitions between parts"
```

#### Problem: Wrong Tempo or Energy
```
SOLUTION PROMPTS:
- Specify exact BPM
- Use multiple tempo descriptors
- Reference feel and groove
- Indicate energy changes

EXAMPLE:
"Slow ballad at exactly 65 BPM, laid-back and contemplative,
half-time feel in verses, slight energy lift in chorus but maintaining slow tempo"
```

#### Problem: Off-Key or Pitchy Vocals
```
SOLUTION PROMPTS:
- Regenerate (Suno's vocal tuning varies per generation)
- Add "professional vocals" or "trained singer"
- Specify "on-pitch vocals"
- Try different vocal style descriptors

EXAMPLE:
"Professional female vocals, on-pitch and emotive,
trained singer quality, clear melodic delivery"
```
