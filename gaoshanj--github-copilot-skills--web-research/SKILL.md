# 网络研究技能

## 何时使用

- 需要收集最新信息
- 需要技术趋势分析
- 需要产品/工具对比
- 需要生成研究报告

## 工作流程

1. **定义研究问题**
2. **搜索相关信息**（通过 MCP）
3. **提取关键数据**
4. **分析和总结**
5. **生成结构化报告**

```markdown

## 执行摘要
...

## 关键发现
...

## 详细分析
...

## 结论和建议
...

## 参考资料
...
```
