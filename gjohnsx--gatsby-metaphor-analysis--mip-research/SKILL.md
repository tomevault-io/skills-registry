# MIP Research Phase

Research scholarly sources on *The Great Gatsby* metaphors and establish the foundation for systematic metaphor identification.

## Your Task

Conduct comprehensive research to prepare for the MIP (Metaphor Identification Procedure) analysis pipeline.

## Research Objectives

### 1. Scholarly Sources
Search for and document:
- Academic analyses of Fitzgerald's metaphor usage in Gatsby
- CMT (Conceptual Metaphor Theory) applications to literary texts
- Jazz Age historical context for symbolic readings
- MIPVU methodology references

### 2. Establish Seed CMT Systems
Identify 3-4 preliminary conceptual metaphor systems expected in the text:
- **DESIRE AS LIGHT** (green light, Daisy's luminosity)
- **MORAL DECAY AS ASH/WASTE** (Valley of Ashes, dust imagery)
- **TIME AS SPACE** (past as place to return to)
- **WEALTH AS DISPLAY** (shirts, parties, possessions)

### 3. MIP Procedure Documentation
Document the exact procedure each annotator will follow:
1. Read passage in full context
2. Identify lexical units
3. Determine contextual meaning (in THIS passage)
4. Determine basic meaning (most concrete, bodily, historically older)
5. Compare: if different and understood via comparison, mark as metaphor
6. Classify MIPVU type (direct, indirect, implicit)
7. Map source → target domains

## Output

Create `mip-analysis/research-notes.md` with:

```markdown
# MIP Research Notes

## Scholarly Context
[Document key sources and findings]

## Seed CMT Systems
[4 systems with expected textual manifestations]

## MIP Procedure for Annotators
[Step-by-step procedure with examples]

## Historical Context
[Jazz Age context relevant to metaphor interpretation]

## Annotation Guidelines
- Target: 8-15 metaphors per chapter
- Focus on thematically significant metaphors
- Include both conventional and novel metaphors
```

## Process

1. Create `mip-analysis/` directory if it doesn't exist
2. Conduct web searches for relevant scholarship
3. Synthesize findings into structured research notes
4. Document the annotator procedure clearly enough for subagents to follow

## Success Criteria

- Research notes contain at least 3 scholarly sources
- 4 seed CMT systems documented with textual expectations
- MIP procedure written as clear, actionable steps
- Ready for `/mip-annotate` phase
