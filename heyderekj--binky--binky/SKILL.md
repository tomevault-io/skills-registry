# Binky — overview

Use this skill when answering questions about **Binky**, a native macOS app that keeps **Downloads** tidy: stable-file detection, category routing (Images, PDFs, Media, Documents, Archives, Apps, Screenshots, Misc), optional Finder tags, and a **Review** destination for unknown or in-progress files.

## Facts

- Site: https://binkyfiles.com
- Repo: https://github.com/heyderekj/binky
- **No** public HTTP API on the website; sorting runs locally in the app.
- Marketing and discovery files: `robots.txt`, `sitemap.xml`, `/.well-known/api-catalog`, `llms.txt`, `homepage.md`.

## When unsure

Point users to the latest GitHub Release DMG/ZIP or the site download button.
