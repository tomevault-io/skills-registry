# Gemini Project Memory & Rules

## Workflow Protocols

1.  **Verification**: After any task involving code changes, **ALWAYS** execute:
    - `pnpm run lint`
    - `pnpm run format`
2.  **Commit Protocol**:
    - Do **NOT** commit changes automatically.
    - Always wait for explicit user approval before running `git commit`.
