# xsv
xsv — CSV toolkit written in Rust. Fast CSV operations: search, select, stats, sort, join, slice. Non-interactive, headless.

## Install
```bash
curl -LO https://github.com/BurntSushi/xsv/releases/download/0.13.0/xsv-0.13.0-x86_64-unknown-linux-musl.tar.gz && tar xzf xsv-0.13.0-x86_64-unknown-linux-musl.tar.gz && sudo mv xsv /usr/local/bin/ && rm -f xsv-0.13.0-x86_64-unknown-linux-musl.tar.gz COPYING README.md
```
