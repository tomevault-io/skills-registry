# 代码检查与优化助手

此 Skill 用于对代码进行深入的检查和优化建议。

## 功能
1. **代码审查**: 检查潜在的 bug、逻辑错误和边界情况。
2. **性能优化**: 识别性能瓶颈，提供优化建议（如算法改进、资源管理）。
3. **代码质量**: 检查代码风格、可读性、命名规范和注释完整性。
4. **最佳实践**: 建议使用现代语言特性和设计模式。

## 使用指南
- 当用户请求“检查代码”、“优化代码”、“Code Review”或类似请求时触发。
- 请先阅读相关文件代码。
- 提供具体的修改建议，最好给出修改前后的代码对比。
- 解释为什么需要这样修改（原理或好处）。
