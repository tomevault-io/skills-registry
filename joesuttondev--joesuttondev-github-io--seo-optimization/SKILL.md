# SEO Optimization Skill

## On-Page SEO Standards

### Title Optimization
- **Length**: 50-60 characters (including spaces)
- Include primary keyword near the beginning
- Make it compelling and click-worthy
- Use power words: "Ultimate", "Easy", "Quick", "Delicious", "Perfect"
- Example: "Easy Vegan Banana Bread | 30-Minute Recipe"

### Meta Description (if applicable)
- **Length**: 150-160 characters
- Include primary keyword
- Include a call-to-action
- Make it descriptive and enticing
- Example: "Discover this easy vegan banana bread recipe. Moist, fluffy, and ready in just 30 minutes. Perfect for breakfast or afternoon tea!"

### URL Structure
- Use hyphens to separate words
- Keep it short and descriptive
- Include primary keyword
- Use lowercase
- Example: `/recipes/easy-vegan-banana-bread`

### Heading Structure
- **H1**: Post title (only one per page)
- **H2**: Main sections (Ingredients, Method, Tips, etc.)
- **H3**: Subsections within main sections
- Use keywords naturally in headings
- Keep headings descriptive and scannable

### Keyword Usage
- **Primary keyword**: Use 3-5 times throughout content
- Include in: title, first paragraph, at least one H2, conclusion
- **LSI keywords**: Include related terms naturally
  - For "vegan cake": plant-based dessert, dairy-free baking, egg-free recipe
- Avoid keyword stuffing - maintain natural flow
- Use variations and synonyms

### Content Structure for SEO

#### Introduction (First Paragraph)
- Include primary keyword in first 100 words
- Hook the reader with a compelling opening
- Preview what they'll learn/get
- Length: 50-150 words

#### Main Content
- **Minimum length**: 
  - Recipes: 500-800 words
  - Lifestyle articles: 1000-1500 words
  - How-to guides: 800-1200 words
- Break content into scannable sections
- Use short paragraphs (2-3 sentences max)
- Include bullet points and numbered lists
- Add relevant internal links (2-4 per post)
- Add external links to authoritative sources (1-2 per post)

#### Conclusion
- Summarize key points
- Include primary keyword
- Add call-to-action (comment, share, subscribe)
- Length: 50-100 words

### Image SEO
- **File names**: descriptive and keyword-rich
  - Good: `vegan-chocolate-cake-slice.jpg`
  - Bad: `IMG_1234.jpg`
- **Alt text**: Descriptive and includes relevant keywords
  - Example: "Slice of moist vegan chocolate cake on a white plate"
- **Dimensions**: Optimize for web (compress images)
- **Format**: Use WebP or JPEG for photos, PNG for graphics

### Internal Linking Strategy
- Link to related recipes or articles
- Use descriptive anchor text (not "click here")
- Link from older posts to newer posts
- Create content clusters around topics
- Example: "Try our [perfect vegan brownies](/recipes/vegan-brownies) next!"

### External Linking
- Link to authoritative sources for:
  - Nutritional information
  - Scientific studies
  - Ingredient sourcing
- Use `rel="nofollow"` for affiliate links
- Open external links in same tab (better for mobile)

### Categories and Tags
- **Categories**: Broad topics (3-5 max)
  - Examples: recipes, lifestyle, nutrition, sustainability
- **Tags**: Specific topics (5-10 per post)
  - Examples: vegan-baking, gluten-free, quick-meals, breakfast

### Schema Markup Considerations
For recipes, ensure metadata supports Schema.org Recipe markup:
- Recipe name
- Author
- Date published/modified
- Preparation time
- Cooking time
- Total time
- Recipe yield (servings)
- Recipe category
- Recipe cuisine
- Keywords
- Nutrition information
- Recipe ingredients
- Recipe instructions

### Content Freshness
- Include publication date
- Update and republish popular posts annually
- Add "Last Updated: DD/MM/YYYY" if updated

## Technical SEO Considerations

### Mobile Optimization
- Ensure responsive design
- Test on multiple devices
- Optimize images for mobile
- Keep load times under 3 seconds

### Readability
- **Flesch Reading Ease**: Aim for 60-70 (accessible to average reader)
- Use short sentences (15-20 words)
- Use simple words when possible
- Break up long paragraphs
- Use transition words

### Loading Speed
- Compress images (max 200KB per image)
- Lazy load images below the fold
- Minimize CSS and JavaScript

## Content Quality Signals

### E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness)
- Share personal experience with recipes
- Cite credible sources
- Include author bio
- Link to social proof (social media)
- Show real photos of finished dishes

### User Engagement
- Encourage comments
- Ask questions throughout content
- Include shareable content
- Add print-friendly recipe cards
- Include save/bookmark functionality

## Local SEO (UK Focus)
- Include UK-specific terminology
- Reference UK seasons for seasonal content
- Use UK location tags where relevant
- Target UK-specific keywords
- Consider UK search trends and interests
