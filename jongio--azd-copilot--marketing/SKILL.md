## Quick Reference

### Landing Page Structure

```
1. Hero Section
   - Headline (value proposition)
   - Subheadline (how it works)
   - CTA button
   - Hero image/demo

2. Problem/Solution
   - Pain points
   - How we solve them

3. Features
   - 3-6 key features with icons
   - Azure integration highlights

4. Social Proof
   - Logos, testimonials, metrics

5. Pricing
   - Tiers with Azure cost context

6. CTA Section
   - Final call to action
   - Trust badges (Azure certified, etc.)
```

### Messaging Framework

| Element | Purpose |
|---------|---------|
| Tagline | 5-7 word memorable phrase |
| Value Prop | One sentence, main benefit |
| Elevator Pitch | 30 seconds, problem + solution |
| Differentiators | 3 unique strengths |
