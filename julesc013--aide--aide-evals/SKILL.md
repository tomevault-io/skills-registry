## Scope

- `evals/**`
- evaluation scaffolds and verification scorekeeping
- matrix-aligned verification posture work

## Trigger

- when a prompt defines smoke checks, regression structure, or compatibility evidence records
- when evaluation outputs must stay aligned with support and capability reporting

## Do Not Use

- Do not use this skill when the task is product implementation instead of verification scaffolding.
- Do not use this skill when the task is only environment acquisition with no evaluation structure.
