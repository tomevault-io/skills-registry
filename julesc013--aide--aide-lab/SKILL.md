## Scope

- `environments/**`
- `labs/**`
- `research/**`
- lab-oriented notes tied to bootability or capture state

## Trigger

- when a prompt is about acquisition state, environment blockers, or reproducible lab setup notes
- when bootability or snapshot status must be recorded without overstating certainty

## Do Not Use

- Do not use this skill when implementing product logic or host adapters.
- Do not use this skill when the task is only inventory normalization with no lab-state component.
