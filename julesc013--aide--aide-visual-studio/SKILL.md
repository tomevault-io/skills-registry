## Scope

- `hosts/microsoft/visual-studio/**`
- Visual Studio lane manifests
- directly related inventory and matrix rows for Visual Studio technologies

## Trigger

- when a prompt targets `com-addin`, `vsix-v1`, `vsix-v2-vssdk`, or `extensibility`
- when Visual Studio lane structure, manifests, or evaluation posture need adjustment

## Do Not Use

- Do not use this skill for Visual Studio for Mac work.
- Do not use this skill for shared-core design that is not specific to Visual Studio lanes.
