# 廣告圖片生成技能

## 目的

根據廣告類型、受眾特徵和文案內容，生成符合平台規範、吸引點擊的廣告圖片。支援合成品牌 logo 和講師照片，自動應用最佳 UI 設計原則，並輸出高品質且檔案大小適中的圖片。

## 支援的廣告類型

根據不同廣告類型，應用不同的視覺設計策略：

1. **課程宣傳**

   - 視覺重點：專業感、學習成果、講師專業形象
   - 配色：知識感（藍、綠、紫系）
   - 元素：書籍、電腦、成長曲線、證書

2. **產品銷售**

   - 視覺重點：產品特寫、使用情境、差異化
   - 配色：依產品屬性（科技藍、健康綠、熱情紅）
   - 元素：產品圖、Before/After、功能圖示

3. **活動宣傳**

   - 視覺重點：時間地點、參與感、限時感
   - 配色：活潑明亮（橘、黃、粉）
   - 元素：日曆、地標、人群、倒數計時

4. **服務推廣**

   - 視覺重點：專業度、信任感、成效展示
   - 配色：穩重專業（深藍、灰、金）
   - 元素：圖表、案例數字、專業工具

5. **品牌建立**
   - 視覺重點：品牌識別、情感連結、記憶點
   - 配色：品牌主色系
   - 元素：品牌符號、slogan、情境故事

## 使用流程

### 步驟 1：確認廣告類型

從使用者請求中識別廣告類型（參考受眾分析報告中的廣告類型）：

- 課程宣傳
- 產品銷售
- 活動宣傳
- 服務推廣
- 品牌建立
- 其他自訂類型

### 步驟 2：詢問素材需求

**必須詢問使用者以下問題**：

1. **是否需要合成 logo？**

   - 若是：請提供 logo 檔案路徑
   - 建議存放位置：`shared/assets/logos/[品牌名稱]_logo.png`
   - Logo 要求：PNG 格式，透明背景，建議尺寸 500x500px 以上

2. **是否需要合成講師照片？**（課程類廣告）

   - 若是：請提供講師照片路徑
   - 建議存放位置：`shared/assets/photos/[講師名稱]_photo.png`
   - 照片要求：PNG/JPG 格式，建議使用專業形象照，建議尺寸 800x800px 以上

3. **參考文案內容**
   - 若已有文案檔案，讀取關鍵資訊
   - 若無，請使用者提供：
     - 主標題（活動名稱或課程名稱）
     - 副標題（可選）
     - 關鍵賣點（1-3 個）

### 步驟 3：研究最佳 UI 設計原則

根據廣告類型，搜尋並應用網路上最吸引點擊的圖片 UI 設計原則：

**通用設計原則**：

1. **視覺層級清晰**

   - 主標題最大最醒目（40-60pt）
   - 副標題次之（24-32pt）
   - 內容文字適中（16-20pt）

2. **色彩對比強烈**

   - 背景與文字有足夠對比度（WCAG AA 標準：4.5:1）
   - 使用補色或對比色吸引注意
   - 避免使用超過 3 種主色

3. **留白充足**

   - 避免資訊過載
   - 重要元素周圍留白
   - 邊緣留白至少 5% 畫面

4. **視覺焦點明確**

   - 使用框線、底色或陰影突顯重點
   - 視線引導：Z 型或 F 型閱讀動線
   - 人臉自動成為視覺焦點

5. **品質感**
   - 高解析度圖片（4K）
   - 避免模糊或失真
   - 專業排版，對齊整齊

**課程類特殊原則**：

- 講師照片占 30-40% 畫面（建立信任感）
- 課程名稱字體粗壯醒目
- 學習成果用圖示化呈現
- 包含「限時」或「名額」標籤（若有）

**產品類特殊原則**：

- 產品圖占 50-60% 畫面
- 功能特點用圖示 + 短文字
- 價格標籤醒目（若強調價格）
- Before/After 對比圖（若強調效果）

**活動類特殊原則**：

- 時間地點最醒目（大字體或框線）
- 活動視覺情境（熱鬧、專業、溫馨）
- 倒數計時或限時標籤
- 參與呼籲（「立即報名」）

### 步驟 4：生成圖片

**技術規格要求**：

- **解析度**：4K（3840 x 4800px）
- **尺寸比例**：4:5（適合 Facebook 和 Instagram）
- **格式**：PNG（支援透明度，方便後製）
- **色彩空間**：sRGB
- **檔案大小目標**：壓縮後小於 1MB

**圖片元素配置（強制規則）**：

1. **主標題位置**：上方 1/3 或中央，字體最大

   - 課程名稱或活動名稱
   - 字體：粗體、無襯線字體（如 Noto Sans TC Bold）
   - 大小：畫面高度的 6-8%
   - 顏色：與背景對比度高

2. **情境元素**：根據文案內容選擇

   - 課程類：學習情境、電腦、筆記、成長
   - 產品類：產品使用情境
   - 活動類：活動氛圍、人群互動
   - 配色與文案情緒一致

3. **主辦單位 logo**：左下角

   - 大小：畫面寬度的 8-12%
   - 位置：距離左下角邊緣 5%
   - 不搶主標題風采
   - 若有白色背景可加淡陰影增加辨識度

4. **講師照片**（若有）：右側或右下

   - 大小：畫面的 30-40%
   - 人像方向面向畫面中央
   - 可用圓形或去背處理
   - 周圍加白邊或陰影增加層次

5. **輔助資訊**：底部或依設計配置
   - 日期時間、地點（活動類）
   - 價格、優惠（產品類）
   - 時數、學習成果（課程類）
   - 字體較小但清晰可讀

### 步驟 5：壓縮圖片

生成圖片後，**必須**自動呼叫壓縮工具：

```python
python utilities/compress_images.py [生成的圖片路徑]
```

**壓縮規格**：

- 目標大小：小於 1MB
- 品質保持：視覺上無明顯差異
- 格式轉換：PNG → 優化的 PNG 或 JPG
- 備份：保留原始 4K 檔案

**壓縮工具使用方式**：

```bash
# 壓縮單張圖片到 1MB 以下
python utilities/compress_images.py input.png

# 指定目標大小（0.5MB）
python utilities/compress_images.py input.png --target-size 0.5

# 壓縮整個資料夾
python utilities/compress_images.py folder/ --recursive
```

### 步驟 6：儲存檔案

**儲存路徑規則**：

```
packages/ad_generator/audience_analysis/[日期時間]_[受眾名稱]/
└── images/
    ├── [平台]_ad_[編號]_4k.png (原始 4K 檔案)
    ├── [平台]_ad_[編號].png (壓縮後檔案，用於上傳)
    └── design_notes.md (設計說明)
```

**檔案命名範例**：

- `facebook_ad_01_4k.png` - Facebook 廣告圖（原始）

## 範例應用

### 範例 1：課程宣傳圖（含講師照片）

**輸入**：

```
廣告類型：課程宣傳
課程名稱：Gemini Gem x NotebookLM 打造行銷小幫手
講師照片：shared/assets/photos/講師_王小明.png
品牌 logo：shared/assets/logos/資展國際_logo.png
受眾分析：packages/ad_generator/audience_analysis/20260204_1430_行銷小編/
```

**執行步驟**：

1. 識別為課程宣傳類型
2. 讀取講師照片和 logo
3. 應用課程類 UI 設計原則
4. 生成 4K 圖片：
   - 主標題：「Gemini Gem x NotebookLM」佔畫面上方
   - 講師照片：右側 30%，專業形象
   - Logo：左下角 10%
   - 情境：AI 學習、數位工具元素
5. 壓縮到 800KB
6. 儲存到 `images/facebook_ad_01.png`

### 範例 2：活動宣傳圖（無照片）

   - 產品視覺化圖示：中央 50%
   - 三大賣點：圖示 + 文字
   - Logo：左下角
   - 情境：科技感、效率感
   - CTA 按鈕：「立即體驗」
4. 壓縮到 950KB
5. 儲存到 `images/facebook_ad_03.png`

## 注意事項

1. **素材準備**

   - Logo 必須是透明背景 PNG
   - 講師照片建議去背或使用專業形象照
   - 所有素材建議存放在 `shared/assets/` 目錄

2. **設計靈感來源**

   - 參考 Facebook Ads Library 高互動廣告
   - 參考 Pinterest 的設計趨勢
   - 參考同類型成功案例

3. **平台適配**

   - Facebook：4:5 或 1:1 比例
   - Instagram：4:5（推薦）或 1:1
   - 若需要其他尺寸，可額外生成

4. **品牌一致性**

   - 使用品牌色系
   - 遵循品牌視覺規範
   - Logo 使用標準版本

5. **文案整合**

   - 圖片文字與文案內容一致
   - 避免資訊重複或衝突
   - 視覺與文案相輔相成

6. **法律合規**

   - 使用授權字體
   - 使用授權圖片素材
   - 避免誤導性視覺表現

7. **⚠️ 不要生成總結報表**：完成圖片生成後，只需生成圖片檔案和 `design_notes.md`（若有需要），**不要**生成 `README.md` 或任何總結報告文件
