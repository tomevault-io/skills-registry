# Decision Matrix

## Workflow
1. **Identifier les options** : lister toutes les alternatives.
2. **Définir les critères** : qu'est-ce qui compte ? (coût, temps, qualité, risque, plaisir…)
3. **Pondérer** : quels critères sont les plus importants ? (poids 1-5)
4. **Évaluer** : noter chaque option sur chaque critère (1-5).
5. **Matrice** : tableau options × critères avec scores pondérés.
6. **Analyse** : quelle option gagne ? Quels sont les trade-offs ?
7. Termine par un résumé clair : "Si tu privilégies X → option A. Si tu privilégies Y → option B."

## Règles
- Ne choisis pas pour l'utilisateur — éclaire le choix.
- Signale les biais possibles (sunk cost, status quo…).
- Demande les critères avant de noter.
