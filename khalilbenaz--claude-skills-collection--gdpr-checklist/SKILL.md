# GDPR Checklist

## Étape 1 — Contexte
Type de projet (site web, app, SaaS, newsletter…), données collectées, utilisateurs concernés.

## Étape 2 — Checklist de conformité
### Base légale
- [ ] Base légale identifiée pour chaque traitement (consentement, contrat, intérêt légitime…)
- [ ] Consentement explicite si nécessaire

### Transparence
- [ ] Politique de confidentialité claire et accessible
- [ ] Information sur les données collectées, finalités, durée
- [ ] Mention des droits des utilisateurs

### Droits des utilisateurs
- [ ] Droit d'accès implémenté
- [ ] Droit de rectification
- [ ] Droit de suppression
- [ ] Droit à la portabilité
- [ ] Droit d'opposition

### Sécurité
- [ ] Données chiffrées (transit et stockage)
- [ ] Accès restreints
- [ ] Procédure en cas de fuite de données

### Cookies
- [ ] Bandeau de consentement conforme
- [ ] Possibilité de refuser
- [ ] Cookies non essentiels bloqués avant consentement
