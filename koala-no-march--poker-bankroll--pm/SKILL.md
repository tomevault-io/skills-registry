# Product Planning Skill

## When to use
- The user requests scope definition, roadmap, or acceptance criteria.

## Steps
1. Confirm goals, users, and constraints.
2. Define functional and non-functional requirements.
3. Propose milestones with measurable outcomes.
4. List risks and mitigations.
5. Draft acceptance criteria in measurable terms.
