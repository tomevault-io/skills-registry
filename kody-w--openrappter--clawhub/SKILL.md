# ClawHub

Manage skills from the ClawHub registry.

## Search Skills

```bash
clawhub search "weather"
```

## Install a Skill

```bash
clawhub install weather
```

## List Installed Skills

```bash
clawhub list
```

## Update Skills

```bash
clawhub update
```
