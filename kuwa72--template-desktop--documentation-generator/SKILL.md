# Skill: Documentation Generator

This skill ensures that project documentation is kept up-to-date as the codebase evolves.

## 📋 Core Documents
- `docs/ai_context.md`: High-level project summary and current status.
- `docs/ui_ux.md`: Visual and behavioral standards.
- `walkthrough.md`: Proof of implementation for specific features.

## 🛠️ Update Procedure
1.  **Analyze Changes**: Review recent file modifications.
2.  **Sync Context**: Update `Current Status` in `ai_context.md` when major milestones are hit.
3.  **Validate Links**: Ensure all cross-references between docs are working.
4.  **Prune Stale Info**: Remove outdated instructions or references.
