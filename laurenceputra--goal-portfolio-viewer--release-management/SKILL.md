# Release Management

Plan and execute releases with clear checklists and documentation.

## Workflow
1. Define release scope and version.
2. Run required checks and update docs.
3. Prepare release notes and artifacts.
4. Monitor post-release and capture follow-ups.

## Output Format
- Release checklist
- Release notes
- Risks/rollbacks
