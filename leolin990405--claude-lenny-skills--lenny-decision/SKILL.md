# Lenny Skills: Decision Making

5 actionable skills for making better decisions.

## Skills

### 1. Running Decision Processes
- Define the decision clearly
- Gather input from stakeholders
- Document the rationale

### 2. Planning Under Uncertainty
- Use scenario planning
- Build in flexibility
- Make reversible decisions quickly

### 3. Evaluating Trade-offs
- Make trade-offs explicit
- Use frameworks consistently
- Accept that perfect doesn't exist

### 4. Post-mortems & Retrospectives
- Focus on learning, not blame
- Identify systemic issues
- Follow through on action items

### 5. Running Effective Meetings
- Have a clear agenda
- Start and end on time
- Document decisions and next steps
