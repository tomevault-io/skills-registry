# Jira Sync

Synchronize local development progress with the Jira issue.

## Usage

```
/jira:sync [issue-key]
```

## Features

- Updates issue status
- Adds progress comments
- Logs work time
- Syncs commits
- Updates labels

## Related Commands

- `/jira:status` - Check current status
- `/jira:commit` - Create smart commit
