# Google Search Skill

## When to Use This Skill
- When you need to find official documentation or guides for a technology
- When searching for API references, tutorials, or troubleshooting steps
- When looking up the latest best practices or release notes
- When a user asks to "google" something or requests external docs

## Prerequisites
- Internet access from the agent or environment
- Clear search query or topic

## Step-by-Step Workflows
1. Receive a search query or documentation request
2. Use Google Search to find the most relevant, official, and up-to-date sources
3. Prioritize results from official domains (e.g., docs.python.org, developer.hashicorp.com, registry.terraform.io, etc.)
4. Return the top 3-5 authoritative links with a short description

## Troubleshooting
- Refine the search query for more specific results
- Add "official documentation" or the technology name to the query
- Check for typos or ambiguous terms
