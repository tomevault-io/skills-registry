# UX-Optimizer Skill

## Propósito
Esta skill está diseñada para analizar, proponer e implementar mejoras en la interfaz de usuario que reduzcan la fricción y aumenten la conversión. Se enfoca en principios de diseño psicológico, accesibilidad y tendencias modernas de e-commerce.

## Capacidades

1.  **Análisis Heurístico**: Evaluar interfaces basándose en las 10 heurísticas de Nielsen (visibilidad del estado, correspondencia con el mundo real, control y libertad, etc.).
2.  **Patrones de Diseño Móvil**: Implementar navegación "Thumb-Zone" (zona del pulgar), áreas táctiles adecuadas (mínimo 44x44px) y flujos simplificados.
3.  **Optimización de Conversión (CRO)**:
    *   Reducción de pasos en el checkout.
    *   Implementación de Cross-Selling y Up-Selling no intrusivo.
    *   Mejora de CTAs (Call to Action) mediante contraste y copy persuasivo.
4.  **Micro-interacciones**: Diseño de feedback visual (animaciones de carga, confirmación de acciones) para mejorar la percepción de velocidad y respuesta.

## Flujo de Trabajo

1.  **Diagnóstico**: Identificar puntos de fricción en el flujo actual (ej. carrito difícil de editar, buscador oculto).
2.  **Propuesta Visual/Funcional**: Describir la mejora (ej. "Añadir barra inferior fija en móvil").
3.  **Implementación Técnica**: Codificar componentes reutilizables (React/Next.js) siguiendo el Design System existente.
4.  **Validación**: Verificar que la mejora funciona en dispositivos móviles y escritorio, y que es accesible.

## Recursos
- Utilizar `framer-motion` para animaciones suaves.
- Priorizar componentes nativos o ligeros para no afectar el rendimiento.
- Seguir las guías de diseño de "Baymard Institute" para e-commerce.
