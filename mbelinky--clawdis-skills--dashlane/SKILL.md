# Dashlane CLI (dcli)

Use `dcli` to access Dashlane vault data securely.

Quick start
- `dcli sync` (first login + sync)
- `dcli p <query>` (passwords)
- `dcli note <filters>` (secure notes)
- `dcli secret <filters>` (secrets)

Notes
- Requires an interactive login on first use.
