# PM Career Accelerator

## When This Skill Activates

Claude uses this skill when:
- Planning career advancement
- Preparing for promotion
- Identifying skill gaps
- Choosing career paths (IC vs management)

## Core Frameworks

### 1. Promotion Criteria

**Typical Levels:**
```
Associate PM → PM → Senior PM → Staff/Principal → Director

Key Dimensions:
- Scope (size of impact)
- Complexity (difficulty of problems)
- Independence (how much guidance needed)
- Leadership (influence on others)
```

### 2. The Promotion Gap Analysis

**Framework:**
```markdown
Current Level → Target Level

GAP ANALYSIS:
- Scope: [current] → [target]
- Impact: [current] → [target]
- Skills: [current] → [target]
- Leadership: [current] → [target]

CLOSE THE GAP:
- High-visibility projects
- Skill development
- Leadership opportunities
```
