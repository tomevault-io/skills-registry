Then describe your optimization task below it.

## What This Skill Does

This skill performs comprehensive Mac optimization including:

- **System Cleanup**: Remove temporary files, cache, and logs
- **Storage Optimization**: Find and remove large unused files
- **Memory Management**: Clear inactive memory and optimize RAM usage
- **Startup Optimization**: Manage launch agents and startup items
- **Permission Repair**: Fix file permissions and disk permissions
- **System Updates**: Check for and install available updates
- **Performance Tuning**: Optimize system settings for better performance

## Safety Features

- **Backup First**: Always creates a backup of critical system files before changes
- **Non-Destructive**: Never removes files without confirmation
- **Rollback Support**: Maintains undo logs for all changes
- **User Approval**: Requires confirmation for potentially risky operations
