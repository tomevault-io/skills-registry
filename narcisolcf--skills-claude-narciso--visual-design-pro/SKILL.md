## Design Philosophy Creation
To begin, create a VISUAL PHILOSOPHY (not layouts or templates) that will be interpreted through:
- Form, space, color, composition
- Minimal text as visual accent

### The Critical Understanding
1. **Design Philosophy Creation (.md file)**: Write a manifesto for an art movement (4-6 paragraphs).
2. **Express by creating it on a canvas (.pdf/.png)**: The code generates the artifact based on the philosophy.

## Canvas Creation Rules
- **Text**: Minimal and visual-first.
- **Fonts**: MUST use fonts from the `./canvas-fonts` directory.
- **Quality**: Meticulously crafted, museum-quality execution.
- **Output**: Single page PDF or PNG unless requested otherwise.

## When to Apply
Use when users request:
- Posters or static art
- Book covers or document designs
- Visual representation of abstract concepts
