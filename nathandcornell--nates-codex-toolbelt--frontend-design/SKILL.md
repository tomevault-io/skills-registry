# Frontend Design Skill

Use this for UI layout, styling, and interaction design tasks.

## Workflow

1. Define a clear visual direction (theme, typography, color palette).
2. Build structure with intentional spacing and hierarchy.
3. Add atmosphere with gradients, shapes, or subtle patterns.
4. Use meaningful animations for key transitions.
5. Validate layout on desktop and mobile.

## Design prompts to answer

- What is the visual story?
- Which typefaces reinforce the brand?
- How does the UI communicate hierarchy at a glance?
- What motion supports comprehension (not noise)?
