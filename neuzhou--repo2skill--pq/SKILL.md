# pq

pq is a Go PostgreSQL driver for database/sql.

## When to Use

- Run `pq` commands
- Run or write tests

## When NOT to Use

- GUI or web-based workflows where CLI is not available
- Projects using Python or JavaScript (different ecosystem)

## Quick Start

### Install

```bash
go install github.com/lib/pq@latest
```

## CLI Commands

- `pq`

## Project Info

- **Language:** Go
