# Forth AI Content Marketing

Generate content for Forth AI's company brand and market presence.

## Scope

This skill is for **company content only**. For personal brand content (Junhua's thought leadership), use the personal-brand skill in the junhua.me workspace.

| Content Type | Use This Skill | Use Personal Brand |
|--------------|----------------|-------------------|
| Product announcements | Yes | No |
| Company blog | Yes | No |
| Case studies | Yes | No |
| Company LinkedIn | Yes | No |
| Thought leadership | No | Yes |
| Personal opinions | No | Yes |
| Founder journey | No | Yes |

## Voice

**Company voice** uses "we" and speaks for Forth AI:
- Professional but not corporate
- Confident about product/vision
- Technical credibility
- Customer-focused

## Content Types

| Type | Purpose | Length |
|------|---------|--------|
| **Product announcement** | New features, releases | 200-400 words |
| **Company blog** | Technical deep-dives, company updates | 800-1500 words |
| **Case study** | Customer success stories | 500-800 words |
| **Company LinkedIn** | Company updates, hiring, milestones | 150-250 words |

## Product Announcement Template

```markdown
# [Feature/Product Name]

[One sentence: What it is and why it matters]

## What's New

[2-3 paragraphs explaining the feature]

## Why We Built This

[Context on customer need or market gap]

## How to Get Started

[Clear next steps]
```

## Company LinkedIn Template

```
[Hook: What we shipped/achieved/learned]

[2-3 short paragraphs with details]

[Call to action: Try it / Read more / Join us]
```

## Quality Checklist

- [ ] Uses "we" (company voice)
- [ ] Focuses on customer value
- [ ] Technically accurate
- [ ] Clear call to action
- [ ] No hype or buzzwords
