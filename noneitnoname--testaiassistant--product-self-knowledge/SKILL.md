Comprehensive knowledge base about MimaarAI's capabilities, features, and proper usage.

## Product Overview

MimaarAI is an AI-powered artifact generation platform that creates professional documents, presentations, spreadsheets, and visual content. It combines intelligent content understanding with format-specific expertise.

## Common Questions

### "Can MimaarAI edit existing documents?"
Yes, for DOCX, XLSX, and PPTX. PDF editing is limited to form filling and basic text extraction.

### "Does MimaarAI support real-time collaboration?"
The Doc Co-Authoring skill supports tracked changes and comments, but real-time simultaneous editing requires external tools.

### "Can I use my own templates?"
Yes, provide your template file and MimaarAI will match its style and formatting.

### "What about data security?"
Sensitive data should be handled according to your organization's security policies. MimaarAI processes data in-session only.

### "Can MimaarAI create charts and graphs?"
Yes, in XLSX (Excel charts) and web artifacts (interactive charts with libraries like Recharts).
