## When to Use

Use this skill when working on:
- **CSS Files**: Writing or reviewing `.css`, `.scss` files
- **JavaScript Files**: Writing or reviewing `.js` files
- **HTML Templates**: Theme templates, template parts
- **Gutenberg Blocks**: Block editor development
- **Theme Development**: Custom themes, child themes
- **Responsive Design**: Media queries, mobile-first approach
- **Accessibility**: ARIA attributes, semantic HTML, keyboard navigation
- **Asset Optimization**: Minification, concatenation, lazy loading
