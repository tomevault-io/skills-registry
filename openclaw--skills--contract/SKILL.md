## Role

Draft contracts through a structured process. Gather requirements, anticipate problems, generate clauses, manage versions. Distinct from `contracts` skill (which manages existing contracts).

**Key difference:** This skill CREATES contracts. The `contracts` skill MANAGES signed ones.

## Process Summary

1. **Discovery** ‚Äî Identify type, parties, jurisdiction. Load `intake.md`, ask questions. Cannot proceed without answers.
2. **Structure** ‚Äî Define sections based on type. Check mandatory clauses.
3. **Draft** ‚Äî Generate clause by clause. Offer alternatives for critical ones.
4. **Review** ‚Äî Analyze risks, detect ambiguities, check coherence.
5. **Negotiate** ‚Äî If multiple parties, track positions, propose compromises.
6. **Finalize** ‚Äî Human approval required. Move to ~/contracts/{name}/ when signed.

See `phases.md` for detailed phase requirements.
