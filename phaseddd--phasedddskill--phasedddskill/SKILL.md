# Tim·影视飓风 写作技能

## 工作流程

撰写 Tim 风格内容涉及以下步骤：

1. 确认写作需求
2. 加载完整风格规范
3. 撰写内容
4. 保存输出

### Step 1：确认写作需求

确认以下信息（用户未提供的主动询问）：

- **主题/话题**：写什么
- **内容类型**：公众号文章 / 视频文案 / 演讲稿 / 其他
- **目标受众**：给谁看
- **大致篇幅**：长文 / 短文 / 不限
- **素材与角度**：是否有特定切入点、参考资料或数据
- **保存路径**：用户指定路径，未指定则默认保存到当前项目根目录

### Step 2：加载风格规范

读取 [references/style-guide.md](references/style-guide.md) 获取完整的 Tim 写作风格规范。

该文件包含：
- 语言节奏与句式（双模式节奏、口语化规范）
- 叙事结构（HKRR 理论、五阶段弧线、升华四步法）
- 核心表达技法（金句构造、类比降维、设问引导、时间尺度切换等）
- 不同场景的风格变化（人文纪实 / 科技评测 / 创业思考 / 哲理展望）
- 情感表达层次
- 格式规范与禁忌清单
- 完整文案范本赏析

### Step 3：撰写内容

严格遵循风格规范撰写。写作过程中重点确保：

- **节奏**：段落式叙述为主体，短句断行仅在需要呼吸时切换
- **升华**：运用四步法（锚定→追问→架桥→落点），段落级和全篇级都需要
- **密度**：遵循 HKRR 时钟理论，每 500 字至少一个兴奋点
- **真诚**：真诚 > 技巧，不为风格而风格

### Step 4：保存输出

- **保存路径**：用户指定路径 > 当前项目根目录
- **文件名格式**：`YYYY-MM-DD-主题关键词.md`
- 保存前展示全文供用户确认或修改
