# Build Deploy

When to use this skill:

- Preparing for production deployment
- Building Docker images
- Running production smoke tests
- Deploying to infrastructure

What this skill does:

1. Builds all affected applications
2. Optimizes bundles for production
3. Runs smoke tests against built artifacts
4. Builds Docker images if configured
5. Verifies environment variables
6. Checks deployment prerequisites
7. Provides deployment commands

Usage:

```
"Build for production"
"Prepare deployment"
"Build and verify production artifacts"
```
