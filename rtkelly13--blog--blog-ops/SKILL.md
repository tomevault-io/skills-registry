# Blog Operations

This skill provides tools to help manage the blog's content and lifecycle.

## Capabilities

- **Create Blog Post**: Scaffolds a new blog post with frontmatter. Can be run interactively.
- **Update Visual Snapshots**: Triggers CI to generate new visual snapshots and downloads them.
- **Verify Deployment**: Checks the health of the current deployment on Vercel and CI status.
- **Audit SEO**: Scans posts for missing SEO tags or summaries.
