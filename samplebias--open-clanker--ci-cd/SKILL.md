# CI/CD

## When to use
- User wants CI/CD status
- Trigger a pipeline
- Check build or deploy

## Steps
1. Identify pipeline and action
2. Use API or CLI if available
3. Report status

## Output format
- Pipeline status (pass/fail)
- Link to run when possible
- Summarize failures
