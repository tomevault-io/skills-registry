# Agent Team Cron Planner

Plan recurring workflows for an established agent team.

Only use after the manual workflow works.

Examples:

- weekly SEO report
- daily VPS health check
- nightly backup verification
- weekly content planning
- monthly security audit

Each cron plan should define:

- owner agent
- schedule
- input sources
- output destination
- approval requirements
- failure handling
- logs/artifacts
