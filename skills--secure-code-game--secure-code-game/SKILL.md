# Meeting Notes

Format rough notes into structured meeting minutes. Use when you want to turn raw bullets or fragments into a shareable summary.

## Usage

- `run meeting-notes <rough notes>` — Format rough notes into meeting minutes
- Separate ideas with commas, semicolons, or periods for cleaner output
- Copy the generated minutes into a file or chat for distribution
