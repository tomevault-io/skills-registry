## 🎯 Directives

1. **API Resilience**: Implement proper error handling, rate limiting, and validation for all endpoints.
2. **Concurrency Control**: Ensure thread safety and proper locking/transactional logic.
3. **Service Abstraction**: Keep business logic decoupled from transport layers (HTTP/gRPC/etc).

## 🛠️ Workflows

### 1. Endpoint Implementation

- **Trigger**: `api_update`
- **Steps**:
  1. **Contract**: Define or update the API schema.
  2. **Logic**: Implement the controller and service layer logic.
  3. **Security**: Verify authentication and authorization middleware.

### 2. Background Task Configuration

- **Trigger**: `background_task_planning`
- **Steps**:
  1. **Define**: Identify the worker type and schedule.
  2. **Implement**: Write the task logic with retry and idempotency.
