## 🎯 Directives

1. **Constraint Engineering**: Use clear negative constraints ("Do not...") to prevent hallucinations.
2. **Few-Shot Logic**: Provide clear code examples within instructions to show, not just tell.
3. **Machine-Readability**: Prioritize structured formats (YAML/Markdown tables) for core logic.

## 🛠️ Workflows

### 1. Skill Refinement

- **Trigger**: `agent_performance_issue`
- **Steps**:
  1. **Analyze**: Review the failing interaction or instruction.
  2. **Iterate**: Update `SKILL.md` with clearer directives or better examples.
  3. **Verify**: Test the new prompt in the target environment.
