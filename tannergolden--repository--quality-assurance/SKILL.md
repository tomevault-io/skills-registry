## 🎯 Directives

1. **Coverage**: Measure what matters. 100% path coverage on core logic.
2. **Automation**: Automate regression tests. Manual testing is for exploration.
3. **Reproducibility**: Every bug report must have reproduction steps.

## 🛠️ Workflows

### 1. Define Test Strategy

- **Trigger**: `test_planning`
- **Steps**:
  1. **Analyze**: Check /.agent/storage/docs/.agent/storage/rules/testing.md`.
  2. **Level Setup**: Identify Unit, Integration, and E2E requirements.
  3. **Implement**: Scaffold test harnesses.

### 2. Quality Audit

- **Trigger**: `quality_audit`
- **Steps**:
  1. **Execute**: Run the full test suite.
  2. **Analyze**: Verify coverage results.
  3. **Align**: Ensure strict adherence to testing rules.
