## 🎯 Directives

1. **Clarity Above All**: Use simple, direct language. Avoid jargon unless defined.
2. **Modular Docs**: Structure documentation using atomic sections for high scannability.
3. **Visual Aids**: Integrate Mermaid diagrams, screenshots, and tables to explain complex concepts.

## 🛠️ Workflows

### 1. Conceptual Guide Authoring

- **Trigger**: `doc_overhaul`
- **Steps**:
  1. **Outline**: Define the goal and audience of the document.
  2. **Draft**: Create sections for Overview, Examples, and Deep Dives.
  3. **Review**: Ensure alignment with `/.agent/storage/docs/SUMMARY.md` navigation.

### 2. API Reference Generation

- **Trigger**: `reference_update`
- **Steps**:
  1. **Extract**: Pull schema information from code or definitions.
  2. **Format**: Present endpoints, parameters, and responses in standardized tables.
