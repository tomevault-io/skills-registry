# 数据分析

帮助用户分析数据，生成可视化图表和统计报告。

## 支持的数据格式

- CSV / TSV 文件
- JSON 数据
- Excel 文件 (.xlsx)
- 数据库查询结果

## 分析步骤

1. 读取并理解数据结构
2. 数据清洗和预处理
3. 执行统计分析
4. 生成可视化图表（使用 matplotlib/seaborn）
5. 输出分析报告

## 输出格式

- 数据概览（行数、列数、数据类型）
- 关键统计指标（均值、中位数、标准差）
- 可视化图表（如适用）
- 分析结论和建议
