# Video Orchestrator: Video-as-Code Briefing

Video-as-Code is the "Remotion Revolution." This skill marks the strategic exit from manual editing suites (Adobe) into a programmatic, agent-led production environment.

## Executive Summary
For the Apex fleet, video is not a static file; it is a React application. Every frame is a component. This shift allows AI agents to build, iterate, and scale video content with surgical precision at a "vibe-prompt" speed.

## Key Capabilities
- **Programmatic Motion**: Using React logic (`interpolate`, `spring`) to define pixel-perfect animations.
- **Vibe-Coding**: Translating natural language descriptions of motion into production-grade code.
- **Wavespeed Integration**: Programmatically triggering 11Labs (Voice) and Nano Banana (Imagery) within the timeline.
- **Infinite Scalability**: Templates that ingest JSON data to generate thousands of personalized variants.

## Strategic Implementation (FLOW Framework)
- **Frame**: Define the "Vibe" and Concept (Length, Format, Audience).
- **Layout**: Design the modular components (Intros, Outros, Lower Thirds).
- **Orchestration**: Iterate throughConcept -> Storyboard -> Detail depth levels.
- **Rendering**: Final export via `npx remotion render`.
- **DNA Alignment**: Ensure visual vibes match the standards in `My-Genie/mygenie_dna.md`.

## Apex AI Standards
- **Progressive Disclosure**: Loading specific documentation (`sequencer.md`, `animation.md`) only when keyframing is active.
- **Clean Root Policy**: Campaign-specific isolation to prevent context pollution.
- **Full Throttle Polish**: Every video must exceed "Safe Bet" standards to achieve a studio-quality look.

## Critical Notes
- **Preview**: Always verify in **Remotion Studio** (`npm run dev`) before rendering.
- **Modular**: Build once, deploy everywhere (TikTok, YouTube, LinkedIn) by swapping composition settings.
