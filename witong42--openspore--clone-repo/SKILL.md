### Usage
`[CLONE_REPO: "https://github.com/user/repo" --path="workspace/my-project"]`

### Arguments
- **repo_url**: The HTTPS URL of the git repository.
- **--path**: (Optional) The target directory inside the workspace. Defaults to `workspace/repo-clone`.
