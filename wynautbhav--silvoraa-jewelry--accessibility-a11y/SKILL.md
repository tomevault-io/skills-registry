# Accessibility Skill

## Core Rules
1. **Keyboard Nav**: Ensure all interactive elements can be reached and activated via Tab/Enter.
2. **Alt Text**: All informative images need `alt` descriptions. Decorative ones need `alt=""`.
3. **Contrast**: Text must have high contrast against the background (check WCAG AA standards).
4. **Labels**: All form inputs must have visible labels or `aria-label` attributes.
