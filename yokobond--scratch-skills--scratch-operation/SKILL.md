# Scratch Operation Skills

This extension provides low-level operation skills for the Scratch editor.

## Available Skills

- [Scratch VM Injection](./skills/scratch-vm-injection/SKILL.md): Programmatically inject blocks into the VM.
- [Scratch Block Dragging](./skills/scratch-block-dragging/SKILL.md): Simulate mouse drag-and-drop to build scripts.
- [Scratch Sprite Library](./skills/scratch-sprite-library/SKILL.md): Add sprites and backdrops from the Scratch built-in library.
- [Custom Costume](./skills/scratch-custom-costume/SKILL.md): Add custom SVG or PNG costumes to sprites programmatically.
- [Scratch Project File](./skills/scratch-project-file/SKILL.md): Download and upload Scratch project .sb3 files.
