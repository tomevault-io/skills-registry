# 财务比率计算器技能

此技能提供全面的财务比率分析，用于评估公司绩效、盈利能力、流动性和估值。

## 功能

计算并解读：
- **盈利能力比率**：ROE、ROA、毛利率、营业利润率、净利润率
- **流动性比率**：流动比率、速动比率、现金比率
- **杠杆比率**：债务权益比、利息保障倍数、偿债保障倍数
- **效率比率**：资产周转率、存货周转率、应收账款周转率
- **估值比率**：P/E、P/B、P/S、EV/EBITDA、PEG
- **每股指标**：EPS、每股账面价值、每股股息

## 使用方法

1. **输入数据**：提供财务报表数据（利润表、资产负债表、现金流量表）
2. **选择比率**：指定要计算的比率，或使用“all”进行综合分析
3. **解读**：该技能将计算比率并提供行业标准解读

## 输入格式

财务数据可以以以下方式提供：
- 包含财务明细项目的 CSV
- 包含结构化财务报表的 JSON
- 关键财务数据的文本描述
- 包含财务报表的 Excel 文件

## 输出格式

结果包括：
- 带数值的计算比率
- 行业基准比较（如果有）
- 趋势分析（如果提供了多个时期）
- 解读和洞察
- 带有格式化结果的 Excel 报告

## 使用示例

“根据所附的财务报表计算这家公司的关键财务比率。”

“如果股价是 50 美元，年度每股收益是 2.50 美元，那么市盈率是多少？”

“使用资产负债表数据分析流动性状况。”

## 脚本

- `calculate_ratios.py`：所有财务比率的主要计算引擎
- `interpret_ratios.py`：提供解读和基准比较

## 最佳实践

1. 在计算之前始终验证数据的完整性
2. 恰当处理缺失值（使用行业平均值或排除）
3. 解读比率时考虑行业背景
4. 包含时期比较以进行趋势分析
5. 标记异常或令人担忧的比率

## 局限性

- 需要准确的财务数据
- 行业基准仅为通用指导原则
- 某些比率可能不适用于所有行业
- 历史数据不能保证未来的表现
